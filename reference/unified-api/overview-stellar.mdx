# Stellar HTTP API

Provides access to all Stellar Horizon endpoints. All usage instructions on the different endpoints are available in [Stellar Docs](https://developers.stellar.org/docs/data/apis/horizon/api-reference)

## Uniblock Query Params

- `chainId`: `stellar` or `stellar-testnet`
- `provider`: select from ones available

## Basic Request

Replace `API_KEY` with actual uniblock api key

<CodeGroup>
  ```bash
  curl --location
  'https://api.uniblock.dev/uni/v1/stellar-http/fee_stats?chainId=stellar' \
  --header 'x-api-key: {{ API_KEY }}' \ --header 'Content-Type:
  application/json' 
  ```
</CodeGroup>
