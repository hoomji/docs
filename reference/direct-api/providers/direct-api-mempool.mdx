For more information regarding the endpoints, please visit Mempools's website: [\<https://mempool.space/docs/api/rest>](https://mempool.space/docs/api/rest).

# Prefix:

No prefix is required for Mempool, an example URL will therefore look like: `https://api.uniblock.dev/direct/v1/Mempool/path/to/endpoint`.

# Example:

<CodeGroup>
  ```bash bash curl --request GET \ --url
  https://api.uniblock.dev/direct/v1/Mempool/v1/services/accelerator/accelerations/history
  \ --header 'accept: application/json' \ --header 'x-api-key:
  YOUR_UNIBLOCK_API_KEY' ```
</CodeGroup>
