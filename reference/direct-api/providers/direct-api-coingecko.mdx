For more information regarding the endpoints, please visit CoinGecko's website: [CoinGecko Docs](https://docs.coingecko.com/reference/introduction).

# Prefix:

No prefix is required for CoinGecko, an example URL will therefore look like: `https://api.uniblock.dev/direct/v1/CoinGecko` + `/[CoinGecko_endpoint]`.

# Example:

<CodeGroup>
  ```bash bash curl --request GET \ --url
  https://api.uniblock.dev/direct/v1/CoinGecko/onchain/simple/networks/eth/token_price/0xc02aaa39b223fe8d0a0e5c4f27ead9083c756cc2
  \ --header 'accept: application/json' \ --header 'x-api-key:
  YOUR_UNIBLOCK_API_KEY' 
  ```
</CodeGroup>
