---
title: 'EXPERIMENTAL_receipt'
description: 'Fetches a receipt by its ID (as is, without a status or execution outcome)'
---
### Parameters

  <ParamField query='chainId' type='string' default='398' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='EXPERIMENTAL_receipt' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={{}} required>
  Returns the original receipt object by ID, without execution outcome or status.
  
  Parameters for <code>EXPERIMENTAL_receipt</code>:
  <ul>
  <li><code>params.receipt_id</code>: Base58-encoded receipt ID. You can find valid IDs via transaction status results.</li>
  </ul>
</ParamField>
