---
title: 'kaia_getTransactionCount'
description: 'Returns the number of transactions sent from an address at the specified block.'
---
### Parameters

  <ParamField query='chainId' type='string' default='1001' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='kaia_getTransactionCount' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={["",0]} placeholder='["",0]' required>
  Parameters: [address, block parameter].
  
  Positional parameters for <code>kaia_getTransactionCount</code>:
  <ul>
  <li><code>params[0]</code>: The address.</li>
  <li><code>params[1]</code>: Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='string'>
  Hexadecimal transaction count. Example: <code>"0x1"</code>.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_getTransactionCount",
    "params": [
      "",
      0
    ]
  }'
```

</RequestExample>

```json Success
  "jsonrpc": "2.0",
```
