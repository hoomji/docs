---
title: 'kaia_nodeAddress'
description: 'Returns the node address derived from the node key.'
---
### Parameters

  <ParamField query='chainId' type='string' default='1001' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='kaia_nodeAddress' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='string'>
  20-byte node address in hex format. Example: <code>"0x9d8a62f656a8d1615c1294fd71e9cfb3e4855a4f"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1001 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_nodeAddress",
    "params": []
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x9d8a62f656a8d1615c1294fd71e9cfb3e4855a4f"
}
```

</ResponseExample>
