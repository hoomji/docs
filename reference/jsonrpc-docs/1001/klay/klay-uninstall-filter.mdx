---
title: 'klay_uninstallFilter'
description: 'Uninstalls the filter identified by the supplied ID.'
---
### Parameters

  <ParamField query='chainId' type='string' default='1001' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='klay_uninstallFilter' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={[""]} placeholder='[""]' required>
  Parameters: [filterId].
  
  Positional parameters for <code>klay_uninstallFilter</code>:
  <ul>
  <li><code>params[]</code>: The filter ID as a hex quantity (e.g., "0x16" for 22).</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='boolean'>
  True if the filter was successfully removed. Example: <code>true</code>.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "klay_uninstallFilter",
    "params": [
      ""
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": true
}
```
