---
title: 'kaia_getTransactionByBlockNumberAndIndex'
description: 'Returns information about a transaction within a block, identified by block number and index.'
---
### Parameters

  <ParamField query='chainId' type='string' default='8217' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='kaia_getTransactionByBlockNumberAndIndex' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={[0,0]} placeholder='[0,0]' required>
  Parameters: [blockNumber, transactionIndex].
  
  Positional parameters for <code>kaia_getTransactionByBlockNumberAndIndex</code>:
  <ul>
  <li><code>params[0]</code>: Block number where this transaction was included. Null when pending.</li>
  <li><code>params[1]</code>: The transaction's index position in the block. Null when pending.</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='string'>
  The transaction.
</ResponseField>
