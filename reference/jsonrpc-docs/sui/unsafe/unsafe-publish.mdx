### Parameters

  <ParamField query='chainId' type='string' default='sui' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='unsafe_publish' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={["0xaabbccddeeff00112233445566778899aabbccdd",[],[],"0x1234567890abcdef","100000"]} placeholder='["0xaabbccddeeff00112233445566778899aabbccdd",[],[],"0x1234567890abcdef","100000"]' required>
  Create an unsigned transaction to publish a Move package.
  
  Positional parameters for <code>unsafe_publish</code>:
  <ul>
  <li><code>params[0]</code>: The Sui address of the transaction signer.</li>
  <li><code>params[1]</code>: List of base64-encoded compiled module bytes for the Move package.</li>
  <li><code>params[2]</code>: List of ObjectIDs representing transitive dependencies.</li>
  <li><code>params[3]</code>: Optional gas object ID to be used. If not provided, one will be selected from signer's coins.</li>
  <li><code>params[4]</code>: Gas budget to be used by this transaction.</li>
  </ul>
</ParamField>
