  Create an unsigned transaction to merge multiple coins into one coin.
  
  Positional parameters for <code>unsafe_mergeCoins</code>:
  <ul>
  <li><code>params[0]</code>: The signer's Sui address.</li>
  <li><code>params[1]</code>: The primary coin object to merge into. This coin will remain.</li>
  <li><code>params[2]</code>: The coin object to be merged. This coin will be destroyed.</li>
  <li><code>params[3]</code>: Optional gas object to be used. Node will auto-select if not provided.</li>
  <li><code>params[4]</code>: Gas budget for the transaction.</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  Unsigned transaction block for merging coins.

    </ResponseField>
    <ResponseField name='txBytes' type='string'>
      Base64-encoded BCS serialized transaction data bytes.
    </ResponseField>
  </Expandable>
</ResponseField>
