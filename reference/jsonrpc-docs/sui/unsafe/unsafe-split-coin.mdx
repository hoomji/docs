### Parameters

  <ParamField query='chainId' type='string' default='sui' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='unsafe_splitCoin' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={["0xaabbccddeeff0011223344556677889900aabbccddeeff001122334455667788","0xcoinobjectabc123456789abcdef",[],"0xgasobject001122334455","100000"]} placeholder='["0xaabbccddeeff0011223344556677889900aabbccddeeff001122334455667788","0xcoinobjectabc123456789abcdef",[],"0xgasobject001122334455","100000"]' required>
  Creates an unsigned transaction to split a coin object into multiple coins.
  
  Positional parameters for <code>unsafe_splitCoin</code>:
  <ul>
  <li><code>params[0]</code>: The transaction signer's Sui address.</li>
  <li><code>params[1]</code>: The coin object ID to split.</li>
  <li><code>params[2]</code>: The list of amounts to split out from the coin.</li>
  <li><code>params[3]</code>: Optional gas object ID. If not specified, the node selects one.</li>
  <li><code>params[4]</code>: The gas budget for the transaction.</li>
  </ul>
</ParamField>
