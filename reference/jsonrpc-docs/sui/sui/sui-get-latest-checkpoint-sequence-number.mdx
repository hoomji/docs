title: 'sui_getLatestCheckpointSequenceNumber'
description: 'Return the sequence number of the latest checkpoint that has been executed'
---
### Parameters

  <ParamField query='chainId' type='string' default='sui' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='sui_getLatestCheckpointSequenceNumber' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='string'>
  Gets the sequence number for the latest checkpoint. Example: <code>"507021"</code>.
</ResponseField>
