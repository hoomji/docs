---
title: 'verifytxoutproof'
description: 'Verifies a Merkle proof produced by gettxoutproof.'
---
### Parameters

  <ParamField query='chainId' type='string' default='bitcoin-testnet4' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='verifytxoutproof' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={["",[]]} placeholder='["",[]]' required>
  Parameters: [proof, txids?].
  
  Positional parameters for <code>verifytxoutproof</code>:
  <ul>
  <li><code>params[0]</code>: Proof data hex.</li>
  <li><code>params[1]</code>: Optional list of transaction ids expected in the proof.</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='array'>
  Transaction ids verified by the proof.
  <Expandable title='items'>
    <ResponseField name='items[]' type='string'>
      32 Bytes - hash of the transaction.
    </ResponseField>
  </Expandable>
</ResponseField>
