---
title: 'getnetworkinfo'
description: 'Returns networking-related information including protocol version, services, and time offsets.'
---
### Parameters

  <ParamField query='chainId' type='string' default='bitcoin' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='getnetworkinfo' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='object'>
  Network status and configuration.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getnetworkinfo",
    "params": []
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {}
}
```

</ResponseExample>
