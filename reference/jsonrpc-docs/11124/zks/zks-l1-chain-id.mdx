title: 'zks_L1ChainId'

---
### Parameters

  <ParamField query='chainId' type='string' default='11124' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='zks_L1ChainId' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='string'>
  The chain ID of the underlying L1 network, in hexadecimal format. Example: <code>"0x1"</code>.
</ResponseField>
