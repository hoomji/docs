---
title: 'avm.getUTXOs'
description: 'Gets the UTXOs that reference a given address. If sourceChain is specified, then it will retrieve the atomic UTXOs exported from that chain to the X Chain.'
---
### Parameters

  <ParamField query='chainId' type='string' default='43113' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='avm.getUTXOs' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={{}} required>
  Parameters for <code>avm.getUTXOs</code>:
  <ul>
  <li><code>params.addresses</code>: A list of X-Chain addresses to fetch UTXOs for.</li>
  <li><code>params.limit</code>: Optional maximum number of UTXOs to return. Defaults to 1024 if omitted or exceeded.</li>
  <li><code>params.startIndex</code>: Pagination index from the previous response's endIndex.</li>
  <li><code>params.sourceChain</code>: Optional source chain ID (e.g., "P" or "C"). Used when fetching atomic UTXOs.</li>
  <li><code>params.encoding</code>: Encoding format for the returned UTXOs. Must be "hex" if specified.</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='numFetched' type='string'>
      Number of UTXOs returned in this response.
    </ResponseField>
    <ResponseField name='utxos' type='array'>
      The list of UTXOs retrieved.
      <Expandable title='items'>
        <ResponseField name='items[]' type='string'>
          A hex-encoded UTXO referencing one of the input addresses.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name='endIndex' type='object'>
      Used as startIndex for paginating the next call.
      <Expandable title='properties'>
        <ResponseField name='address' type='string'>
          The last address included in the result.
        </ResponseField>
        <ResponseField name='utxo' type='string'>
          The last UTXO ID included in the result.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name='sourceChain' type='string'>
      The source chain if this was an atomic UTXO query.
    </ResponseField>
    <ResponseField name='encoding' type='string'>
      The encoding used for the returned UTXOs.
    </ResponseField>
  </Expandable>
</ResponseField>
