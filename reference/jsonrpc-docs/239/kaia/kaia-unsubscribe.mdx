---
title: 'kaia_unsubscribe'
description: 'Cancels a websocket subscription previously created with kaia_subscribe.'
---
### Parameters

  <ParamField query='chainId' type='string' default='239' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='kaia_unsubscribe' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={[""]} placeholder='[""]' required>
  Parameters: [subscriptionId].
  
  Positional parameters for <code>kaia_unsubscribe</code>:
  <ul>
  <li><code>params[]</code>: Subscription identifier.</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='boolean'>
  True if the subscription was successfully cancelled. Example: <code>true</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=239 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_unsubscribe",
    "params": [
      ""
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": true
}
```

</ResponseExample>
