---
title: 'num_unconfirmed_txs'
description: 'Get data on unconfirmed transactions.'
---
### Parameters

  <ParamField query='chainId' type='string' default='2222' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='num_unconfirmed_txs' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='object'>
  The unconfirmed transactions information.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=2222 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "num_unconfirmed_txs",
    "params": []
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "n_txs": "31",
    "total": "82",
    "total_bytes": "19974"
  }
}
```

</ResponseExample>
