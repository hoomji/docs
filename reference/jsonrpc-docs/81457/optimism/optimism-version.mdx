### Parameters

  <ParamField query='chainId' type='string' default='81457' required>

  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='optimism_version' required>
    JSON-RPC method name.
  </ParamField>

  The software version. Example: <code>"v0.0.0-"</code>.
</ResponseField>
