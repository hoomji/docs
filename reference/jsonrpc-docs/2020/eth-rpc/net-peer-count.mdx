---
title: 'net_peerCount'
description: 'Returns number of peers currenly connected to the client.'
---
### Parameters

  <ParamField query='chainId' type='string' default='2020' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='net_peerCount' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='integer'>
  Quantity - Integer of the number of connected peers Example: <code>"0x2"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=2020 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "net_peerCount",
    "params": []
  }'
```

</RequestExample>

  "jsonrpc": "2.0",
