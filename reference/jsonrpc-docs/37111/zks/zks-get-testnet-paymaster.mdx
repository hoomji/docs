---
title: 'zks_getTestnetPaymaster'
description: 'Returns the address of the testnet paymaster available on testnets and enables paying fees in ERC-20 compatible tokens.'
---
### Parameters

  <ParamField query='chainId' type='string' default='37111' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='zks_getTestnetPaymaster' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='string'>
  The address of the testnet paymaster. Example: <code>"0x8f0ea1312da29f17eabeb2f484fd3c112cccdd63"</code>.
</ResponseField>

```bash Curl
    "params": []
  }'
```
