---
title: 'getVersionInfo'
description: 'Returns version information about the RPC service.'
---
### Parameters

  <ParamField query='chainId' type='string' default='stellar-soroban' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='getVersionInfo' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='object'>
  Software/version metadata for the Soroban RPC service.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=stellar-soroban \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getVersionInfo",
    "params": []
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "name": "soroban-rpc",
    "version": "x.y.z",
    "git": {
      "commit": "abcdef",
      "branch": "main"
    }
  }
}
```

</ResponseExample>
