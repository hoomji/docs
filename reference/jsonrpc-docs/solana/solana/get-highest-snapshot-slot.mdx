---
title: 'getHighestSnapshotSlot'
description: 'Returns the highest slot information that the node has snapshots for. This will find the highest full snapshot slot, and the highest incremental snapshot slot based on the full snapshot slot, if there is one.'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive' 
---
### Parameters

  <ParamField query='chainId' type='string' default='solana' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='getHighestSnapshotSlot' required>
    JSON-RPC method name.
  </ParamField>

This method does not require any parameters.

### Returns

<ResponseField name='result' type='object'>
  Highest snapshot slot information including full and incremental snapshot slots.
  <Expandable title='properties'>
    <ResponseField name='full' type='integer'>
      Highest full snapshot slot.
    </ResponseField>
    <ResponseField name='incremental' type='integer'>
      Highest incremental snapshot slot.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getHighestSnapshotSlot",
    "params": []
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "full": 100,
    "incremental": 110
  }
}
```

</ResponseExample>
