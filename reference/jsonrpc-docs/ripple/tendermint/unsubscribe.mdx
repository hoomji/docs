---
title: 'unsubscribe'
description: 'Unsubscribe from event on Websocket.'
---
### Parameters

  <ParamField query='chainId' type='string' default='ripple' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='unsubscribe' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={{}}>
  Parameters for <code>unsubscribe</code>:
  <ul>
  <li><code>params.query</code>: Optional query string used to unsubscribe from a specific subscription. If omitted, all subscriptions for the client will be removed.</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "unsubscribe",
    "params": {}
  }'
```

</RequestExample>
