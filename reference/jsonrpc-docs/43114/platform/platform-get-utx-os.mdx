---
title: 'platform.getUTXOs'
description: 'Gets the UTXOs that reference a given set of addresses.'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive' 
---
### Parameters

  <ParamField query='chainId' type='string' default='43114' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='platform.getUTXOs' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={{}} required>
  Parameters for <code>platform.getUTXOs</code>:
  <ul>
  <li><code>params.addresses</code>: A list of addresses to fetch UTXOs for.</li>
  <li><code>params.limit</code>: Maximum number of UTXOs to fetch (max: 1024).</li>
  <li><code>params.startIndex</code>: Pagination start index.</li>
  <li><code>params.sourceChain</code>: Source chain ID (e.g. "X" or "C") to retrieve atomic UTXOs.</li>
  <li><code>params.encoding</code>: Encoding for the UTXOs (currently supports "hex").</li>
  </ul>
</ParamField>
