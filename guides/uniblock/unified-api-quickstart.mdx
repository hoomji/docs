Estimated time to complete this guide: < 2 minutes

Here, you'll find a basic step-by-step process to test our Unified API. This guide assumes that you've already created a project. Let's get started.

<Steps>
  <Step title="Copy your API key">
    Navigate to your project overview. Copy your API key.

    ![](/images/docs/a6be502f75e449dbc028f53e7b8509b9eb48787a8fcf67095934701422386ed0-image.png)
  </Step>

  <Step title="Navigate to the API Tester">
    Head to our explorer by clicking on the `Endpoints` section on the left nav-bar, then click on `API Tester`.

    <Frame caption='Project Overview Page -> Endpoints -> API Tester'>
      <img src='/images/docs/9626bd59828718ef05897ce110408080c6a1de5f788427aa3decacd386cb6fdc-image.png' />
    </Frame>
  </Step>

  <Step title="Open the explorer">
    It will open the explorer a new tab for you. You will see this page.

    ![](/images/docs/d861b9d17236b271bbd21af14148454b3bc373a9e36a1ae755dfb754bf5ae297-image.png)
  </Step>

  <Step title="Authorize your API key">
    Authorize your key on the explorer by hitting the `Authorize` button. Then paste your Uniblock API key to it and hit the `Authorize` button.

    ![](/images/docs/3781610903394d9d9b2eb04b2bb129c48e63057d610a4d6c7248d6f04f8c1616-image.png) ![](/images/docs/984d72cf9dfca51a2a491aff2c9c3301d1250fb5377f9f1463c8daa4c778831b-image.png) ![](/images/docs/9d5628b5cce636eb624cd859becbd6309338d89414997019b92c3fffa6b59de9-image.png)
  </Step>

  <Step title="Select the endpoint">
    We will try calling the `/token/metadata` endpoint. Select the endpoint and click the `Try it out` button.

    ![](/images/docs/776b232dc9e2f6d25eacd3106f75b408cfe08d209dd2e67b9468d2ebc82051c9-image.png)
  </Step>

  <Step title="Execute the request">
    There are example values filled in the required fields by default. Alternatively you can enter your own input parameter values. Then click on the `Execute` button to call the endpoint.

    ![](/images/docs/bacc04e42b46cbd3393f9d14625019ea90dc6eae7574b9a0ad93852411d33127-image.png)
  </Step>

  <Step title="View the response">
    Here is the successful response.

    ![](/images/docs/b985d3df5a5c6efc999424610553cd597f3932200b5c68567667e00222506981-image.png)

    Nice job! You've made an API call with our Unified API! ðŸŽ‰
  </Step>
</Steps>
