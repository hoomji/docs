### Priority List (Fallback List):

- Navigate to the `Endpoints` -> `Unified API` page.
- Select an endpoint.
- Go to the `Fallback` tab.
- Turn on `Enable Fallback List`.
- Adjust the provider fallback list on an individual endpoint basis. You can drag and drop each integration to order it from top to bottom.
