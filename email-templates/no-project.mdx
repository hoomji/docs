# No Project After 24h Email

**Trigger:** 24–48h after signup  
**Criteria:** User has 0 projects  
**Audience:** Individual user  
**Goal:** Push first activation step

## Email Template

**Subject:** Need help creating your first project?
**Preheader:** Create a project in 2 minutes to unlock your API key.

---

Hi {{first_name}},

We noticed you haven't created a project yet — no worries, we're here to help you get started.

### Why create a project?

Projects unlock your API keys and let you start making requests. There's no commitment required, and you can create one in under 2 minutes.

### Here's how it works

1. Go to your [dashboard](https://dashboard.unblock.com)
2. Click "Create project"
3. Give it a name
4. Get your API key

That's it. You'll be ready to make your first request.

### Common questions

**Not sure where to start?**

**Just testing things out?**  
Perfect — our free tier is designed for exactly that. No credit card needed.

If you're stuck or have questions, just reply to this email. We're happy to walk you through it.
