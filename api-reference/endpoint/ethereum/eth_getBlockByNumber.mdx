### Parameters

This method requires **two** positional parameters:

- `blockNumber` (string | integer): Block identifier. Can be a block number or a tag: `"earliest"`, `"latest"`, or `"pending"`.
- `fullTransactions` (boolean): If `true`, returns full transaction objects. If `false`, returns only transaction hashes.

</ParamField>

  Request identifier.
</ParamField>

<ParamField body='jsonrpc' type='string' required prefill='2.0'>

</ParamField>

<ParamField body='method' type='string' required prefill='eth_blockNumber'>

</ParamField>

<ParamField body='params' type='array' prefill={[]}>

</ParamField>
