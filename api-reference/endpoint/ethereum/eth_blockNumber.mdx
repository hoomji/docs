### Parameters

This method does not require any parameters.

### Returns

<ResponseField body='result' type='string'>
  The latest block number encoded as a hexadecimal string (e.g.,{' '}
  <code>"0x15536ee"</code>).
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://YOUR_UNIBLOCK_BASE_URL/json-rpc?chainId=1 \
  --header 'content-type: application/json' \
  --data '{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": []
}'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": []
}
```

</RequestExample>
