### Parameters

This method does not require any parameters.

### Returns

<ResponseField body='result' type='string'>
  The address of the ZKsync Era main contract. Example: <code>"0x32400084c286cf3e17e7b677ea9583e60a000324"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=232 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "zks_getMainContract",
    "params": []
  }'
```
