### Parameters

Parameters for ots_getBlockTransactions: [defaultBlock].

- `params[0]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField body='result' type='object'>
  Block object with full transactions and their receipts, optimized for pagination. Includes truncated input data and simplified receipts.
</ResponseField>

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "ots_getBlockTransactions",
