### Parameters

The parameters for ots_getBlockDetailsByHash: [blockHash].

- `params[]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.

### Returns

<ResponseField body='result' type='object'>
  Modified block object with additional fields and stripped transaction list.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "ots_getBlockDetailsByHash",
```
