---
title: 'eth_getProof'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [address, storageKeys, blockParameter]

- `params[0]` (string): The 20-byte address to check for balance, nonce, and storage.
- `params[1]` (array): Array of 32-byte storage keys (hex) to be included in the proof.
- `params[2]` (string): Block number or tag ("latest", "earliest", "pending", "safe", "finalized").

### Returns

<ResponseField body='result' type='object'>
  Account details and Merkle proofs for storage verification.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getProof",
    "params": [
      "0x7F0d15C7FAae65896648C8273B6d7E43f58Fa842",
      [
        "0x56e81f171bcc55a6ff8345e692c0f86e5b48e01b996cadc001622fb5e363b421"
```
