### Parameters

Parameters for eth_getTransactionCount: [address, defaultBlock].

- `params[0]` (string): The address.
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField name='result' type='string'>
  The number of transactions sent from this address. Example: <code>"0x1b"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=5373 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getTransactionCount",
    "params": [
      "0x407d73d8a49eeb85d32cf465507dd71d507100c1"
    ]
  }'
```
