### Parameters

This method does not accept any parameters.

### Returns

<ResponseField body='result' type='array'>
  Array of available compiler names (typically empty in most modern clients).
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=300 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getCompilers",
    "params": [
      null
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",

```
