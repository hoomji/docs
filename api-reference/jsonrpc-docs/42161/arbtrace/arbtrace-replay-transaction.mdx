---
title: 'arbtrace_replayTransaction'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

An array containing the transaction hash and a list of trace types to include.

- `params[0]` (string): Transaction hash (32-byte hex).
- `params[1]` (array): List of trace types to include in the output.

### Returns

<ResponseField body='result' type='object'>
  Execution trace details for the replayed transaction.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=42161 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "arbtrace_replayTransaction",
    "params": [
      "0xa3a97b9d1de234efb2ca9374c5918f9a7fc2adff4102aff686df01b6a3dc74f7",
      [
        "trace"
      ]
    ]
  }'
```
