---
title: 'arbtrace_replayBlockTransactions'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Array with the block number and trace types to return.

- `params[0]` (string): Block number to replay (hex format).
- `params[1]` (array): Trace types to include for each transaction in the block.

### Returns

<ResponseField body='result' type='array'>
  Array of trace objects, one per transaction in the block.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=42161 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "arbtrace_replayBlockTransactions",
    "params": [
      "0xb48c3c",
      [
        "trace"
      ]
    ]
  }'
```

```json JSON-RPC

```

</RequestExample>
