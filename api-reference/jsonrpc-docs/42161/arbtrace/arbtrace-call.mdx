---
title: 'arbtrace_call'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

An array containing the transaction call object, trace types, and block tag/number.

- `params[0]` (object): Transaction call object for simulation.
- `params[1]` (array): Trace types to return: trace, vmTrace, or stateDiff.
- `params[2]` (string): Block number (hex) or tag ("latest", "pending").

### Returns

<ResponseField body='result' type='object'>
  Traces of the simulated call execution.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=42161 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "arbtrace_call",
    "params": [
      {},
      [
        "trace"
      ],
      "latest"
    ]
  }'
```

