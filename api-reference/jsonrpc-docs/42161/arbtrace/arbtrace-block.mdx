---
title: 'arbtrace_block'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Array containing a single block number (as hex) or tag (e.g., "latest").

- `params[]` (string): The block number (hex) or tag ("latest", "pending").

### Returns

<ResponseField body='result' type='array'>
  List of trace objects for all transactions in the block.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=42161 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "arbtrace_block",
    "params": [
      "0xb48c3c"
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "arbtrace_block",
  "params": [
    "0xb48c3c"
  ]
}
```

</RequestExample>
