### Parameters

The parameters for method debug_traceBlock: [blockRlp, traceConfig].

- `params[0]` (string): The RLP encoded block.
- `params[1]` (object): Optional configuration object passed to tracing RPC methods.

### Returns

<ResponseField name='result' type='array'>
  Full stack trace.
</ResponseField>
