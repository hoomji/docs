---
title: 'zks_getBlockDetails'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

- `params[]` (integer): The number of the L2 block to fetch details for.

### Returns

<ResponseField body='result' type='object'>
  Detailed information about a ZKsync L2 block.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=11124 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "zks_getBlockDetails",
    "params": [
      140599
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "zks_getBlockDetails",
  "params": [
    140599
  ]
}
```

</RequestExample>
