### Parameters

Parameters for eth_getTransactionByHash: [transactionHash].

- `params[0]` (string): 32 Bytes - Hash of the transaction.

### Returns

<ResponseField body='result' type='oneOf'>
  Transaction response object. Can be a legacy transaction (pre-EIP-2718) or an AccessList transaction (EIP-2930).
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=11124 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getTransactionByHash",
    "params": [
      "0xbfb3bb785c60ee4f93aa6e506f6f5d5fc5173301f537d856cac59d91e7a0ce09"
    ]
  }'
```
