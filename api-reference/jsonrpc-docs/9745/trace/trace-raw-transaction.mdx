### Parameters

Parameters for the trace_rawTransaction method: [transactionData, traceType].

- `params[0]` (string): Data - Raw transaction data.
- `params[1]` (array): Types of traces to collect: "vmTrace", "trace", and/or "stateDiff".

### Returns

<ResponseField name='result' type='object'>
  Object - Block traces.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=9745 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "trace_rawTransaction",
    "params": [
      "",
      []
    ]
  }'
```
