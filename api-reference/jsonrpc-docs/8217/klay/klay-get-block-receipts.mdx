---
title: 'klay_getBlockReceipts'
openapi: 'POST /json-rpc'
playground: 'interactive'

### Parameters

Parameters: [blockHashOrNumber].

- `params[0]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.
- `params[1]` (oneOf): Block number where this transaction was included. Null when pending.

### Returns

<ResponseField body='result' type='array'>
  Array of transaction receipts for the block.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=8217 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "klay_getBlockReceipts",
    "params": [
      "",
      0
    ]
  }'
```
