### Parameters

Parameters: [address, block parameter].

- `params[0]` (string): The address.
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField body='result' type='boolean'>
  True if the account has been created. Example: <code>true</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=8217 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "klay_accountCreated",
    "params": [
      "0x1b3c8c1cdb4e6b6b8819345bf3c6d9c926f55f90",
      "latest"
    ]
  }'
```
