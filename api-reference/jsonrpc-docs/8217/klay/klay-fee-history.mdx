### Parameters

Parameters: [blockCount, newestBlock, rewardPercentiles].

- `params[0]` (string): Hex-encoded number of blocks in the request range (max 1024).
- `params[1]` (oneOf): Block number where this transaction was included. Null when pending.
- `params[2]` (array): Priority fee percentiles to return (0-100).
