### Parameters

Parameters: [address, position, block parameter].

- `params[0]` (string): The address.
- `params[1]` (string): Hex-encoded storage slot index.
- `params[2]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField name='result' type='string'>
  Hex-encoded storage value. Example: <code>"0x0"</code>.
</ResponseField>

```bash Curl
  --header 'content-type: application/json' \

    "id": 1,
    "jsonrpc": "2.0",
    "method": "klay_getStorageAt",

      "",

  }'
```
