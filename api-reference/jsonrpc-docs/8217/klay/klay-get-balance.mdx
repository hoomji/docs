### Parameters

  <ParamField query='chainId' type='string' default='8217' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='klay_getBalance' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={["0x1b3c8c1cdb4e6b6b8819345bf3c6d9c926f55f90","latest"]} placeholder='["0x1b3c8c1cdb4e6b6b8819345bf3c6d9c926f55f90","latest"]' required>
  Parameters: [address, block parameter].
  
  Positional parameters for <code>klay_getBalance</code>:
  <ul>
  <li><code>params[0]</code>: The address.</li>
  <li><code>params[1]</code>: Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".</li>
  </ul>
</ParamField>

### Returns

<ResponseField name='result' type='string'>

</ResponseField>
