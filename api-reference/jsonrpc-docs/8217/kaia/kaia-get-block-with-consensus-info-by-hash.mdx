### Parameters

Parameters: [blockHash].

- `params[]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.

### Returns

<ResponseField name='result' type='object'>
  Block data combined with consensus information.
  <Expandable title='properties'>
    <ResponseField name='block' type='object'>
      Block information similar to kaia_getBlockByHash.
    </ResponseField>
    <ResponseField name='committee' type='array'>
      Addresses of committee members.
      <Expandable title='items'>
        <ResponseField name='items[]' type='string'>
          The address.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name='proposer' type='string'>
      Address of the committee member who proposed the block.
    </ResponseField>
  </Expandable>
</ResponseField>
