---
title: 'kaia_getBlockByHash'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [blockHash, fullTransactions].

- `params[0]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.
- `params[1]` (boolean): If true, includes full transaction objects; otherwise only hashes.

### Returns

<ResponseField body='result' type='object'>
  A Kaia block object. Null if the block cannot be found.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=8217 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_getBlockByHash",
    "params": [
      "0x9a4e12c1b8ef6f2f726755efaf3f0b0f3788c2d9d11b9bd65d9f8c17e7a019d2",
      true
    ]
  }'
```
