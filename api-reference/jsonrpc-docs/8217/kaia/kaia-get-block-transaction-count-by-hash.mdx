title: 'kaia_getBlockTransactionCountByHash'
openapi: 'POST /json-rpc'

### Parameters

Parameters: [blockHash].

- `params[]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.

### Returns

<ResponseField body='result' type='string'>
  Hexadecimal quantity representing the number of transactions in the block. Example: <code>"0x10"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=8217 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_getBlockTransactionCountByHash",
    "params": [
      "0x9a4e12c1b8ef6f2f726755efaf3f0b0f3788c2d9d11b9bd65d9f8c17e7a019d2"
    ]
  }'
```
