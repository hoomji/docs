### Parameters

This method does not require any parameters.

### Returns

<ResponseField body='result' type='string'>
  20-byte node address in hex format. Example: <code>"0x9d8a62f656a8d1615c1294fd71e9cfb3e4855a4f"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=8217 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_nodeAddress",
    "params": []
  }'
```
