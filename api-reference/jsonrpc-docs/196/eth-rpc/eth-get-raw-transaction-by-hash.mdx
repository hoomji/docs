### Parameters

Parameters: [transactionHash]

- `params[]` (string): The hash of the transaction (32-byte hex string).

### Returns

<ResponseField name='result' type='string'>
  Raw transaction data in hexadecimal format, including sender, recipient, gas, data, and signature. Example: <code>"0xf86c018504a817c8008252089471c7656ed63d30e271b7db6f5cb94d10d5c2b7b3880de0b6b3a76400008025a0ba4681b..."</code>.
</ResponseField>
