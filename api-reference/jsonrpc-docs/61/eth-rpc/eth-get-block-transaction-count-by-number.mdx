### Parameters

Parameters for eth_getBlockTransactionCountByNumber: [blockNumber].

- `params[0]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField name='result' type='string'>
  Integer of the number of transactions in this block. Example: <code>"0x1b"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=61 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getBlockTransactionCountByNumber",
    "params": [
      "0x1b4"
    ]
  }'
```

</RequestExample>
