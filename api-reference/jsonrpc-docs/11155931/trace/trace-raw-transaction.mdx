### Parameters

Parameters for the trace_rawTransaction method: [transactionData, traceType].

- `params[0]` (string): Data - Raw transaction data.
- `params[1]` (array): Types of traces to collect: "vmTrace", "trace", and/or "stateDiff".

### Returns

<ResponseField body='result' type='object'>
  Object - Block traces.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "trace_rawTransaction",

  }'
```
