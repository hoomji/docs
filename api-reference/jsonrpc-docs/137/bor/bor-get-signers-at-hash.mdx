### Parameters

Parameters for bor_getSignersAtHash: [blockHash]

- `params[]` (string): 32-byte block hash (hex-encoded, 0x-prefixed).

### Returns

<ResponseField name='result' type='array'>
  List of validator addresses (signers) for the specified block hash.
  <Expandable title='items'>
    <ResponseField name='items[]' type='string'>
      Validator signer address (20-byte hex string).
    </ResponseField>
  </Expandable>
</ResponseField>
