### Parameters

The parameters for method unconfirmed_txs: [limit].

- `params[]` (integer): The maximum number of unconfirmed transactions to return.

### Returns

<ResponseField name='result' type='object'>
  The list of transactions.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=137 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "unconfirmed_txs",
    "params": [
      10
    ]
  }'
```
