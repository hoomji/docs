### Parameters

The parameters for method consensus_params: [height].

- `params[]` (integer): The height to return.

### Returns

<ResponseField name='result' type='object'>
  Consensus params information.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=137 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "consensus_params",
    "params": [
      1
    ]
  }'
```
