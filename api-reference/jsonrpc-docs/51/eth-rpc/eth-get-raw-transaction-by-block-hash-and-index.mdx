playground: 'none'

### Parameters

Parameters: [blockHash, transactionIndex]

- `params[0]` (string): The hash of the block containing the transaction.
- `params[1]` (string): The index position of the transaction in the block (hexadecimal).

### Returns

<ResponseField name='result' type='string'>
  Raw transaction data as a hexadecimal string, including the signature. Example: <code>"0xf86b018504a817c8008252089471c7656ed63d30e271b7db6f5cb94d10d5c2b7b3880de0b6b3a76400008026a0ba4681b..."</code>.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{

    "jsonrpc": "2.0",

      "0x7e4aea365093a183b0df5f003f26144ef04ec8a6e7f4910356590c5fc7b1671f",

  }'
```
