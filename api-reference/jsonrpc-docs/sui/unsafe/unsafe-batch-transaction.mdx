### Parameters

Create an unsigned batched transaction.

- `params[0]` (string): The transaction signer's Sui address.
- `params[1]` (array): List of transaction request parameters.
- `params[2]` (string): Optional gas object ID to be used. If not provided, one will be chosen by the node.
- `params[3]` (string): Gas budget (maximum gas units that can be used).
- `params[4]` (string): Transaction builder mode. Optional.

### Returns

<ResponseField body='result' type='object'>
  Unsigned batched transaction block.
</ResponseField>
