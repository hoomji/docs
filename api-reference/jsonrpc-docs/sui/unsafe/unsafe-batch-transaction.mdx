### Parameters

Create an unsigned batched transaction.

- `params[0]` (string): The transaction signer's Sui address.
- `params[1]` (array): List of transaction request parameters.
- `params[2]` (string): Optional gas object ID to be used. If not provided, one will be chosen by the node.
- `params[3]` (string): Gas budget (maximum gas units that can be used).
- `params[4]` (string): Transaction builder mode. Optional.

### Returns

<ResponseField name='result' type='object'>
  Unsigned batched transaction block.
  <Expandable title='properties'>
    <ResponseField name='gas' type='array'>
      Gas objects used in this transaction.
    </ResponseField>
    <ResponseField name='inputObjects' type='array'>
      List of objects used as input in the transaction.
    </ResponseField>
    <ResponseField name='txBytes' type='string'>
      BCS serialized transaction data bytes (base64).
    </ResponseField>
  </Expandable>
</ResponseField>
