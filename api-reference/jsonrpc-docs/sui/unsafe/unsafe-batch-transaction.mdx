  Positional parameters for <code>unsafe_batchTransaction</code>:
  <ul>
  <li><code>params[0]</code>: The transaction signer's Sui address.</li>
  <li><code>params[1]</code>: List of transaction request parameters.</li>
  <li><code>params[2]</code>: Optional gas object ID to be used. If not provided, one will be chosen by the node.</li>
  <li><code>params[3]</code>: Gas budget (maximum gas units that can be used).</li>
  <li><code>params[4]</code>: Transaction builder mode. Optional.</li>
  </ul>
</ParamField>
