### Parameters

Creates an unsigned transaction to split a coin object into multiple equal-size coins.

- `params[0]` (string): The transaction signer's Sui address.
- `params[1]` (string): The coin object ID to split.
- `params[2]` (string): The number of equal-sized coins to split into.
- `params[3]` (string): Optional gas object ID. If not specified, the node selects one.
- `params[4]` (string): The gas budget for the transaction.

### Returns

<ResponseField body='result' type='object'>
  The unsigned transaction block that performs the coin split into equal-size parts.
</ResponseField>
