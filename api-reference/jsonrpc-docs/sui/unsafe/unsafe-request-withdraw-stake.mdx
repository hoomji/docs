  Creates an unsigned transaction to withdraw staked SUI.
  
  Positional parameters for <code>unsafe_requestWithdrawStake</code>:
  <ul>
  <li><code>params[0]</code>: The Sui address of the transaction signer.</li>
  <li><code>params[1]</code>: The object ID of the staked SUI to withdraw.</li>
  <li><code>params[2]</code>: Optional gas object ID. If omitted, a gas object will be selected automatically.</li>
  <li><code>params[3]</code>: The gas budget for the transaction.</li>
  </ul>
</ParamField>
