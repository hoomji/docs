### Parameters

Creates an unsigned transaction to withdraw staked SUI.

- `params[0]` (string): The Sui address of the transaction signer.
- `params[1]` (string): The object ID of the staked SUI to withdraw.
- `params[2]` (string): Optional gas object ID. If omitted, a gas object will be selected automatically.
- `params[3]` (string): The gas budget for the transaction.

### Returns

<ResponseField body='result' type='object'>
  The unsigned transaction block that performs the stake withdrawal.
</ResponseField>
