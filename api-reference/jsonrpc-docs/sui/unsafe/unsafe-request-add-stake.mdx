### Parameters

Create an unsigned transaction to stake SUI to a validator.

- `params[0]` (string): The Sui address of the transaction signer.
- `params[1]` (array): List of Coin<SUI> object IDs to be staked.
- `params[2]` (string): Amount to stake. If omitted, the full balance of all coins will be used.
- `params[3]` (string): The validator's Sui address.
- `params[4]` (string): Optional gas object ID. If not provided, a gas object will be auto-selected.
- `params[5]` (string): Gas budget for the transaction.

### Returns

<ResponseField body='result' type='object'>
  The unsigned transaction block for the stake operation.
</ResponseField>
