  Create an unsigned transaction to stake SUI to a validator.
  
  Positional parameters for <code>unsafe_requestAddStake</code>:
  <ul>
  <li><code>params[0]</code>: The Sui address of the transaction signer.</li>
  <li><code>params[1]</code>: List of Coin<SUI> object IDs to be staked.</li>
  <li><code>params[2]</code>: Amount to stake. If omitted, the full balance of all coins will be used.</li>
  <li><code>params[3]</code>: The validator's Sui address.</li>
  <li><code>params[4]</code>: Optional gas object ID. If not provided, a gas object will be auto-selected.</li>
  <li><code>params[5]</code>: Gas budget for the transaction.</li>
  </ul>
</ParamField>
