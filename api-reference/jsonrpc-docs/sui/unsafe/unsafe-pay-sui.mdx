### Parameters

Create an unsigned transaction to send SUI coins to multiple recipients with defined amounts.

- `params[0]` (string): The Sui address of the transaction signer.
- `params[1]` (array): List of SUI coin object IDs to be used (including gas coin).
- `params[2]` (array): List of recipient Sui addresses.
- `params[3]` (array): List of amounts to send to each recipient (in the same order).
- `params[4]` (string): Gas budget to be used by this transaction.

### Returns

<ResponseField body='result' type='object'>
  The unsigned transaction block for a SUI coin payment.
</ResponseField>
