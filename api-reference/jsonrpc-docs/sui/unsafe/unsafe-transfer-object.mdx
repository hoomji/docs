### Parameters

Create an unsigned transaction to transfer an object to a recipient. The object must allow public transfers.

- `params[0]` (string): The signer's Sui address.
- `params[1]` (string): The ID of the object to be transferred.
- `params[2]` (string): Optional gas object ID. If not specified, the node picks one automatically.
- `params[3]` (string): The gas budget for the transaction.
- `params[4]` (string): The recipient's Sui address.

### Returns

<ResponseField body='result' type='object'>
  Unsigned transaction bytes and metadata for the object transfer.
</ResponseField>
