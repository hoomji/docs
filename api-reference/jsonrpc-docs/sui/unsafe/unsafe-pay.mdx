### Parameters

Create an unsigned transaction to send Coin<T> to multiple recipients.

- `params[0]` (string): The signer's Sui address.
- `params[1]` (array): The Sui coin object IDs to use for payment.
- `params[2]` (array): List of recipient Sui addresses.
- `params[3]` (array): Amounts to send to each recipient, in the same order.
- `params[4]` (string): Optional gas object ID. If not provided, the node will select one.
- `params[5]` (string): Gas budget for the transaction.

### Returns

<ResponseField body='result' type='object'>
  Unsigned transaction block for sending multiple payments.
</ResponseField>
