### Parameters

  <ParamField query='chainId' type='string' default='sui' required>
    Unique identifier for a blockchain network.

  <ParamField body='id' type='number' default={1} required>

    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={["0xabcde1234567890abcde1234567890abcde1234567890abcde1234567890abcd",["0xcoin1","0xcoin2"],["0xrecipient1","0xrecipient2"],["10000","5000"],"0xgasobject","100000"]} placeholder='["0xabcde1234567890abcde1234567890abcde1234567890abcde1234567890abcd",["0xcoin1","0xcoin2"],["0xrecipient1","0xrecipient2"],["10000","5000"],"0xgasobject","100000"]' required>
  Create an unsigned transaction to send Coin<T> to multiple recipients.
  
  Positional parameters for <code>unsafe_pay</code>:
  <ul>
  <li><code>params[0]</code>: The signer's Sui address.</li>
  <li><code>params[1]</code>: The Sui coin object IDs to use for payment.</li>
  <li><code>params[2]</code>: List of recipient Sui addresses.</li>
  <li><code>params[3]</code>: Amounts to send to each recipient, in the same order.</li>
  <li><code>params[4]</code>: Optional gas object ID. If not provided, the node will select one.</li>
  <li><code>params[5]</code>: Gas budget for the transaction.</li>
  </ul>
</ParamField>
