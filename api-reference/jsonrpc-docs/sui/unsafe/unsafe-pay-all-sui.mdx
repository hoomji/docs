### Parameters

Create an unsigned transaction to send all SUI coins to a recipient.

- `params[0]` (string): The Sui address of the transaction signer.
- `params[1]` (array): List of SUI coin object IDs to be used, including the gas coin.
- `params[2]` (string): The recipient's Sui address.
- `params[3]` (string): The gas budget for the transaction.

### Returns

<ResponseField body='result' type='object'>
  Unsigned transaction block for sending all SUI coins to a recipient.
</ResponseField>
