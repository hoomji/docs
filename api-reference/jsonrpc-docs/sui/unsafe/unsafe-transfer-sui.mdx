### Parameters

Create an unsigned transaction to send SUI coin to a recipient. The SUI object is used as both transfer and gas coin.

- `params[0]` (string): The signer's Sui address.
- `params[1]` (string): The SUI coin object to use in this transaction.
- `params[2]` (string): Gas budget for the transaction.
- `params[3]` (string): The recipient's Sui address.
- `params[4]` (string): Optional transfer amount. If not provided, all balance is transferred minus gas cost.

### Returns

<ResponseField body='result' type='object'>
  Unsigned transaction block containing base64 encoded BCS bytes and transaction metadata.
</ResponseField>
