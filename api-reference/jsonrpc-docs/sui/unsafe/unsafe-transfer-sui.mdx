  Create an unsigned transaction to send SUI coin to a recipient. The SUI object is used as both transfer and gas coin.
  
  Positional parameters for <code>unsafe_transferSui</code>:
  <ul>
  <li><code>params[0]</code>: The signer's Sui address.</li>
  <li><code>params[1]</code>: The SUI coin object to use in this transaction.</li>
  <li><code>params[2]</code>: Gas budget for the transaction.</li>
  <li><code>params[3]</code>: The recipient's Sui address.</li>
  <li><code>params[4]</code>: Optional transfer amount. If not provided, all balance is transferred minus gas cost.</li>
  </ul>
</ParamField>
