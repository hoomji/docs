### Parameters

Paramters for method suix_getCoinMetadata: [coinType].

- `params[]` (string): Coin type

### Returns

<ResponseField name='result' type='object'>
  The metadata for the coin type in the request
  <Expandable title='properties'>
    <ResponseField name='decimals' type='integer'>
      Number of decimal places the coin uses.
    </ResponseField>
    <ResponseField name='description' type='string'>
      Description of the token
    </ResponseField>
    <ResponseField name='iconUrl' type='string'>
      URL for the token logo
    </ResponseField>
    <ResponseField name='id' type='string'>
      Object ID for the CoinMetadata object
    </ResponseField>
    <ResponseField name='name' type='string'>
      Name for the token
    </ResponseField>
    <ResponseField name='symbol' type='string'>
      Symbol for the token
    </ResponseField>
  </Expandable>
</ResponseField>

```bash Curl
curl --request POST \

```
