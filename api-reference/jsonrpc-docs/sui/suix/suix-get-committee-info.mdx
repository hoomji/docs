### Parameters

Parameters for suix_getCommitteeInfo: [epoch]. The epoch is optional; if omitted, the latest epoch is used.

- `params[]` (string): The epoch number as a BigInt for uint64, represented as a string (e.g., "1234567890").

### Returns

<ResponseField body='result' type='object'>
  Committee information for the specified epoch.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "suix_getCommitteeInfo",
    "params": [
      5000
    ]
  }'
```
