### Parameters

### Returns

<ResponseField body='result' type='array'>

</ResponseField>

<RequestExample>

```bash Curl

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "suix_getAllBalances",
```
