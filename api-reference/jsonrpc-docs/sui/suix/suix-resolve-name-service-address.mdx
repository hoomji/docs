### Parameters

Parameters for suix_resolveNameServiceAddress: [name]

- `params[]` (string): The name to resolve.

### Returns

<ResponseField body='result' type='string'>
  The resolved Sui address (as a base-58 encoded string). Example: <code>"0x6710024f81dd33ab6833482ee8034e779a48e6ef635c7f856df4905022458bfb"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "suix_resolveNameServiceAddress",
    "params": [
      "example.sui"
    ]
  }'
```
