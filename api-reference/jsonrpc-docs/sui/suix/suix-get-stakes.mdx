### Parameters

Parameters for suix_getStakes: [owner]. The owner is a SuiAddress (base-58 encoded string).

- `params[]` (string): The owner's Sui address.

### Returns

<ResponseField body='result' type='array'>
  Array of DelegatedStake objects.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "suix_getStakes",
    "params": [
      "0xa6f5a7953a75dc632e696cabe60560522a017bf2fb0bd930d1ec22c06f1ee4e4"
    ]
  }'
```
