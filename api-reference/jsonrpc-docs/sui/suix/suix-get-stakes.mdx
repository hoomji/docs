### Parameters

Parameters for suix_getStakes: [owner]. The owner is a SuiAddress (base-58 encoded string).

- `params[]` (string): The owner's Sui address.

### Returns

<ResponseField name='result' type='array'>
  Array of DelegatedStake objects.
  <Expandable title='items'>
    <ResponseField name='items[]' type='object'>
      Object representing a delegated stake.
      <Expandable title='properties'>
        <ResponseField name='validator' type='string'>
          The Sui address of the validator receiving the stake.
        </ResponseField>
        <ResponseField name='stake_amount' type='string'>
          The amount staked (u64, as a string representing a BigInt).
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

curl --request POST \

    ]
