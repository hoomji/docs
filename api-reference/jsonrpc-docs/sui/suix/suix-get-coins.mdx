### Parameters

Parameters for suix_getCoins: [ownerAddress, coinType, cursor, limit].

- `params[0]` (string): The owner's Sui address
- `params[1]` (string): Optional type name for the coin
- `params[2]` (string): Optional paging cursor
- `params[3]` (integer): Maximum number of items per page
