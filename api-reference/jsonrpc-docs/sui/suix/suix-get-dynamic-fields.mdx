### Parameters

Parameters for suix_getDynamicFields.

- `params[0]` (string): The ID of the parent object.
- `params[1]` (string): Optional paging cursor. If provided, the query will start from the item after this cursor.
- `params[2]` (integer): Maximum number of items per page. Defaults to QUERY_MAX_RESULT_LIMIT if not specified.

### Returns

<ResponseField body='result' type='object'>
  DynamicFieldPage object containing dynamic field info and pagination cursor.
</ResponseField>
