### Parameters

Get all events emitted by the transaction.

- `params[]` (string): The transaction digest.

### Returns

<ResponseField body='result' type='object'>
  The JSON-RPC result.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "sui_getEvents",
    "params": [
      "11a72GCQ5hGNpWGh2QhQkkusTEGS6EDqifJqxr7nSYX"
    ]
  }'
```
