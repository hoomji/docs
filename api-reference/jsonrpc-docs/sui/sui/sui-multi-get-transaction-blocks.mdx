### Parameters

Retrieve multiple transaction blocks by digest.

- `params[0]` (array): List of transaction digests.
- `params[1]` (object): Optional settings to configure the returned transaction block fields.

### Returns

<ResponseField body='result' type='array'>
  Array of detailed transaction block responses.
</ResponseField>
