### Parameters

Retrieve object data at a specific version (if retained by the node).

- `params[0]` (string): Object ID in hex format.
- `params[1]` (integer): The version of the object to retrieve.
- `params[2]` (object): Options for customizing the object response fields.

### Returns

<ResponseField body='result' type='object'>
  Response for a past object retrieval attempt.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "sui_tryGetPastObject",
    "params": [
      "0x11af4b844ff94b3fbef6e36b518da3ad4c5856fa686464524a876b463d129760",
```
