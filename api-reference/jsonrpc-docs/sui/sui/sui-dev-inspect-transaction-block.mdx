  Run a transaction in dev-inspect mode for debugging and analysis purposes.
  
  Positional parameters for <code>sui_devInspectTransactionBlock</code>:
  <ul>
  <li><code>params[0]</code>: Sender address (SuiAddress).</li>
  <li><code>params[1]</code>: BCS encoded TransactionKind (base64).</li>
  <li><code>params[2]</code>: Gas price used for gas cost estimation.</li>
  <li><code>params[3]</code>: Epoch number to use for dev inspection.</li>
  <li><code>params[4]</code>: Optional DevInspectArgs including gas budget, gas objects, skip checks, etc.</li>
  </ul>
</ParamField>
