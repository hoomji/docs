### Parameters

Run a transaction in dev-inspect mode for debugging and analysis purposes.

- `params[0]` (string): Sender address (SuiAddress).
- `params[1]` (string): BCS encoded TransactionKind (base64).
- `params[2]` (string): Gas price used for gas cost estimation.
- `params[3]` (string): Epoch number to use for dev inspection.
- `params[4]` (object): Optional DevInspectArgs including gas budget, gas objects, skip checks, etc.
