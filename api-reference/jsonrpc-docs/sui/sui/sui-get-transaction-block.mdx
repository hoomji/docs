### Parameters

Parameters to fetch transaction block information.

- `params[0]` (string): Transaction digest (base58 string).
- `params[1]` (object): Optional transaction block response configuration options.

### Returns

<ResponseField body='result' type='object'>
  Detailed transaction block response object.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "sui_getTransactionBlock",
    "params": [
      "Hay2tj3GcDYcE3AMHrej5WDsHGPVAYsegcubixLUvXUF",
      {}
    ]
  }'
```
