### Parameters

Execute a signed transaction and optionally wait for effects or local execution.

- `params[0]` (string): Base64-encoded BCS serialized transaction data (without type tag).
- `params[1]` (array): Array of base64-encoded signatures.
- `params[2]` (object): Optional transaction response options.
- `params[3]` (string): Execution mode indicating how long to wait for confirmation.
