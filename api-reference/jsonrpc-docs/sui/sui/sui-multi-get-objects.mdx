### Parameters

Fetch metadata for multiple Sui objects by ID.

- `params[0]` (array): Array of Object IDs to retrieve.
- `params[1]` (object): Optional flags to customize object data returned.

### Returns

<ResponseField body='result' type='array'>
  Array of SuiObjectResponse items, each representing one object.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "sui_multiGetObjects",
    "params": [
      [],
      {}
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "sui_multiGetObjects",
  "params": [
    [],
    {}
  ]
}
```

</RequestExample>
