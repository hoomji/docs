### Parameters

  <ParamField query='chainId' type='string' default='sui' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='sui_getCheckpoint' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={["1000"]} placeholder='["1000"]' required>
  Parameters to fetch checkpoint data by digest or sequence number.
  
  Positional parameters for <code>sui_getCheckpoint</code>:
  <ul>
  <li><code>params[]</code>: Checkpoint ID - can be a checkpoint digest or sequence number.</li>
  </ul>
</ParamField>
