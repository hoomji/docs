title: 'sui_getCheckpoint'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters to fetch checkpoint data by digest or sequence number.

- `params[]` (string): Checkpoint ID - can be a checkpoint digest or sequence number.

### Returns

<ResponseField body='result' type='object'>
  The JSON-RPC result.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "sui_getCheckpoint",
    "params": [
      "1000"
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "sui_getCheckpoint",
  "params": [
    "1000"
  ]
}
```

</RequestExample>
