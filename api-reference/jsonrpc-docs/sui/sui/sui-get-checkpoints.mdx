### Parameters

Fetch a paginated list of checkpoints.

- `params[0]` (string): An optional paging cursor (sequence number).
- `params[1]` (integer): Maximum number of items to return per page.
- `params[2]` (boolean): Whether to return results in descending order.

### Returns

<ResponseField body='result' type='object'>
  The JSON-RPC result.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "sui_getCheckpoints",
    "params": [
      "1004",
      4,
      false
    ]
  }'
```
