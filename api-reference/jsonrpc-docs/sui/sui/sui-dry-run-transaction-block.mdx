### Parameters

Base64-encoded BCS transaction block data.

- `params[]` (string): BCS-encoded transaction block (TransactionKind).

### Returns

<ResponseField body='result' type='object'>
  Simulated transaction block execution result, including effects, events, and gas usage.
</ResponseField>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=sui \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "sui_dryRunTransactionBlock",
    "params": [
      "AAACACB7qR3cfnF89wjJNwYPBASHNuwz+xdG2Zml5YzVxnftgAEAT4LxyFh7mNZMAL..."

  }'
```
