### Parameters

Returns the minimum amount of tokens required to validate or delegate on the given Subnet.

- `subnetID` (string): Optional ID of the subnet to query. Defaults to the Primary Network if omitted.

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='minValidatorStake' type='string'>
      Minimum stake required to become a validator on the subnet.
    </ResponseField>
    <ResponseField name='minDelegatorStake' type='string'>
      Minimum stake required to delegate to a validator.
    </ResponseField>
  </Expandable>
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \

    "method": "platform.getMinStake",
    "params": {}
  }'
```
