---
title: 'platform.getUTXOs'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive'
---
### Parameters

- `addresses` (array): A list of addresses to fetch UTXOs for.
- `limit` (integer): Maximum number of UTXOs to fetch (max: 1024).
- `startIndex` (object): Pagination start index.
- `sourceChain` (string): Source chain ID (e.g. "X" or "C") to retrieve atomic UTXOs.
- `encoding` (string): Encoding for the UTXOs (currently supports "hex").
