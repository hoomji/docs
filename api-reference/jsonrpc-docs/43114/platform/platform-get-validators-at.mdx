title: 'platform.getValidatorsAt'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive'

### Parameters

- `height` (oneOf): The P-Chain block height to query validator set for.
- `subnetID` (string): Subnet ID to query validator set for. If omitted, queries the Primary Network.

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='validators' type='object'>
      A mapping from validator NodeID to their stake weight.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=43114 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "platform.getValidatorsAt",
    "params": {}
  }'
```

</RequestExample>

```json Success
  "jsonrpc": "2.0",
  "id": 1,

```

</ResponseExample>
