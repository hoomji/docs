### Parameters

- `tx` (string): The serialized transaction bytes, typically hex-encoded and prefixed with 0x.
- `encoding` (string): The encoding format of the transaction. Can only be "hex" when provided.

  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='txID' type='string'>
      The transaction ID assigned by the network.
    </ResponseField>
  </Expandable>
</ResponseField>
