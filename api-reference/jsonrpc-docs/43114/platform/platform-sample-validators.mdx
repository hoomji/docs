### Parameters

- `size` (integer): Number of validators to sample from the specified Subnet.
- `subnetID` (string): Optional Subnet ID to sample validators from. Defaults to the Primary Network.

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='validators' type='array'>
      A list of sampled validator NodeIDs.
      <Expandable title='items'>
        <ResponseField name='items[]' type='string'>
          The NodeID of a validator.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
