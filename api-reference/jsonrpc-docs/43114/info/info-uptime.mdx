### Parameters

Optional object specifying a particular subnet ID to evaluate uptime against.

- `subnetID` (string): Optional subnet ID to query uptime within. If omitted, uptime is evaluated on the Primary Network.

### Returns

<ResponseField body='result' type='object'>
  Uptime metrics based on stake-weighted observations from the network.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=43114 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "info.uptime",
    "params": {}
  }'
```

