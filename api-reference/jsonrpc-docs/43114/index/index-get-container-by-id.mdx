---
title: 'index.getContainerByID'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive'
---
### Parameters

Parameters to identify and retrieve the container.

- `id` (string): The container ID to look up.
- `encoding` (string): The encoding format for the byte output. Must be "hex".

### Returns

<ResponseField name='result' type='object'>
  The container data and metadata.
  <Expandable title='properties'>
    <ResponseField name='id' type='string'>
      The unique ID of the container.
    </ResponseField>
    <ResponseField name='bytes' type='string'>
      Hex-encoded byte representation of the container.
    </ResponseField>
    <ResponseField name='timestamp' type='string'>
      Timestamp when the node accepted this container.
    </ResponseField>
    <ResponseField name='encoding' type='string'>
      The encoding format of the `bytes` field (always "hex").
    </ResponseField>
    <ResponseField name='index' type='string'>
      The sequential index of this container in the accepted container list.
    </ResponseField>
  </Expandable>
</ResponseField>
