### Parameters

Encoding configuration for the returned data.

- `encoding` (string): Specifies the encoding format. Must be "hex".

### Returns

<ResponseField body='result' type='object'>
  Details of the last accepted container on the chain.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=43114 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "index.getLastAccepted",
    "params": {
      "encoding": "hex"
    }
  }'
```
