---
title: 'avm.getAssetDescription'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive'
---
### Parameters

- `assetID` (string): The asset ID to query. For example, the AVAX asset ID differs by network (Mainnet: FvwEAhmxK..., Testnet: U8iRqJoi...).

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='assetID' type='string'>
      The ID of the queried asset.
    </ResponseField>
    <ResponseField name='name' type='string'>
      The human-readable name of the asset.
    </ResponseField>
    <ResponseField name='symbol' type='string'>
      The short symbol used to represent the asset.
    </ResponseField>
    <ResponseField name='denomination' type='integer'>
      Determines display formatting. E.g., denomination=9 means 1000000000 base units = 1 displayed unit.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=43114 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "avm.getAssetDescription",
    "params": {}
  }'
```
