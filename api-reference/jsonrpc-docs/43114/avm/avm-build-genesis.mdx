### Parameters

- `networkID` (integer): The network ID for which the genesis state will be generated.
- `genesisData` (object): JSON description of each asset in the genesis state.
- `encoding` (string): The encoding format of the returned genesis bytes.

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='bytes' type='string'>
      The encoded byte representation of the genesis state.
    </ResponseField>
    <ResponseField name='encoding' type='string'>
      The encoding format used.
    </ResponseField>
  </Expandable>
</ResponseField>
