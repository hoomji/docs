---
title: 'avm.getUTXOs'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive'
---
### Parameters

- `addresses` (array): A list of X-Chain addresses to fetch UTXOs for.
- `limit` (integer): Optional maximum number of UTXOs to return. Defaults to 1024 if omitted or exceeded.
- `startIndex` (object): Pagination index from the previous response's endIndex.
- `sourceChain` (string): Optional source chain ID (e.g., "P" or "C"). Used when fetching atomic UTXOs.
- `encoding` (string): Encoding format for the returned UTXOs. Must be "hex" if specified.

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='numFetched' type='string'>
      Number of UTXOs returned in this response.
    </ResponseField>
    <ResponseField name='utxos' type='array'>
      The list of UTXOs retrieved.
      <Expandable title='items'>
        <ResponseField name='items[]' type='string'>
          A hex-encoded UTXO referencing one of the input addresses.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name='endIndex' type='object'>
      Used as startIndex for paginating the next call.
      <Expandable title='properties'>
        <ResponseField name='address' type='string'>
          The last address included in the result.
        </ResponseField>
        <ResponseField name='utxo' type='string'>
          The last UTXO ID included in the result.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name='sourceChain' type='string'>
      The source chain if this was an atomic UTXO query.
    </ResponseField>
    <ResponseField name='encoding' type='string'>
      The encoding used for the returned UTXOs.
    </ResponseField>
  </Expandable>
</ResponseField>
