### Parameters

- `addresses` (array): Array of addresses to search UTXOs for.
- `limit` (integer): Maximum number of UTXOs to fetch (default: 1024)
- `startIndex` (object): Start index for pagination.
- `sourceChain` (string): Source chain name (e.g., X, P)
- `encoding` (string): Optional encoding for returned UTXOs

### Returns

<ResponseField body='result' type='object'>
  The JSON-RPC result.
</ResponseField>
