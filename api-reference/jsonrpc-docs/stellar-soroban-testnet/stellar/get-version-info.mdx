### Parameters

This method does not require any parameters.

### Returns

<ResponseField body='result' type='object'>
  Software/version metadata for the Soroban RPC service.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=stellar-soroban-testnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getVersionInfo",
    "params": []
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "getVersionInfo",
  "params": []
}
```

</RequestExample>
