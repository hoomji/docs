---
title: 'getTransactions'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

- `startLedger` (integer): Inclusive starting ledger sequence to list from.
- `order` (string): Sort order by creation time.
- `account` (string): Optional account to filter transactions by.
- `pagination` (object): Pagination controls.

### Returns

<ResponseField body='result' type='object'>
  Horizon transactions page.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=stellar-soroban-testnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getTransactions",
    "params": {
      "startLedger": "LEDGER_SEQUENCE",
      "pagination": {
        "limit": 10
      }
    }
  }'
```
