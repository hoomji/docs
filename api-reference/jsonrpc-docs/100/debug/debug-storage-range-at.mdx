### Parameters

Parameters: [blockHash, txIdx, contractAddress, keyStart, maxResult]

- `params[0]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.
- `params[1]` (integer): Index of the transaction within the block.
- `params[2]` (string): 20 Bytes - The contract address created, if the transaction was a contract creation. Null otherwise.
- `params[3]` (string): Starting key (32-byte hex string) used for filtering storage or entries.
- `params[4]` (integer): Maximum number of results to return.

### Returns

<ResponseField body='result' type='array'>
  The array of storage items.
</ResponseField>
