### Parameters

Parameters: [filePath, seconds].

- `params[0]` (string): Destination file for the execution trace.
- `params[1]` (integer): Duration to trace in seconds.

### Returns

<ResponseField name='result' type='boolean'>
  True if tracing completed successfully. Example: <code>true</code>.
</ResponseField>

    "id": 1,
    "jsonrpc": "2.0",
    "method": "debug_goTrace",

      "",

    ]
  }'
