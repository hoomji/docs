---
title: 'dag_getHeads'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

- `params[]` (string): Epoch tag to query (e.g., "pending" or a specific epoch number as a string).

### Returns

<ResponseField body='result' type='array'>
  List of event IDs with no descendants in the specified epoch.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=250 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "dag_getHeads",
    "params": [
      "pending"
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "dag_getHeads",
  "params": [
    "pending"
  ]
}
```

</RequestExample>
