### Parameters

- `params[0]` (integer): The block number in which the message was emitted.
- `params[1]` (string): The address that sent the message.
- `params[2]` (string): The keccak256 hash of the message being proved.
- `params[3]` (integer): Optional index in the block of the L1Messenger event log. Defaults to the first message if omitted.
