### Parameters

Parameters for eth_getBalanceValues: [address, blockTag]

- `params[0]` (string): Address to query (20 bytes, hex-encoded).
- `params[1]` (string): Block tag (e.g., "latest", "earliest", or a hex block number).

### Returns

<ResponseField name='result' type='object'>
  Detailed balance values for an Ethereum address.
  <Expandable title='properties'>
    <ResponseField name='liquidBalance' type='string'>
      ETH balance available for transfer (QUANTITY, 64 bits).
    </ResponseField>
    <ResponseField name='stakedBalance' type='string'>
      Amount staked (QUANTITY, 64 bits).
    </ResponseField>
    <ResponseField name='pendingRewards' type='string'>
      Pending rewards not yet claimed (QUANTITY, 64 bits).
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=81457 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getBalanceValues",
    "params": [
      "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
      "latest"
    ]
  }'
```
