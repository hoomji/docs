---
title: 'dag_getEventPayload'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

- `params[0]` (string): The full event ID (hex-encoded 32 bytes) or short event ID.
- `params[1]` (boolean): If true, returns full transaction objects; otherwise, returns transaction hashes.

### Returns

<ResponseField body='result' type='object'>
  An event object, or null if no event was found.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=57054 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "dag_getEventPayload",
    "params": [
      "0x00033bbf00000719167a3b2c445bd3e53e9c35d16cf52ac24852317b9605d07e",
      true
    ]
  }'
```
