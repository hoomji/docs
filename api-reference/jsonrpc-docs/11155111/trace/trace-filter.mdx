### Parameters

Parameters for method trace_filter: [filterObject]

- `params[]` (object): Filter options used to create a new filter for logs. Topics are order-dependent. Nulls or nested arrays are used to match multiple possibilities.

### Returns

<ResponseField body='result' type='array'>
  Array - Traces matching given filter
</ResponseField>
