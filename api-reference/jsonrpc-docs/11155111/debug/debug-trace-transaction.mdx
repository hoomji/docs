### Parameters

Parameters for method debug_traceTransaction: [txHash, traceConfig].

- `params[0]` (string): 32 Bytes - Hash of the transaction.
- `params[1]` (object): Optional configuration object passed to tracing RPC methods.

### Returns

<ResponseField name='result' type='object'>
  The ExecutionResult of the transaction.
</ResponseField>
