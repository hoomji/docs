### Parameters

Parameters for method debug_traceBlockByNumber: [blockNumber, traceConfig]

- `params[0]` (oneOf): Block number where this transaction was included. Null when pending.
- `params[1]` (object): Optional configuration object passed to tracing RPC methods.

### Returns

<ResponseField name='result' type='array'>
  Full stack trace.
</ResponseField>
