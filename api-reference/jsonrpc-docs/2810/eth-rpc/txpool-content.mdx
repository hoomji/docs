### Parameters

This method does not require any parameters.

### Returns

<ResponseField name='result' type='object'>
  Transaction pool content separated by pending and queued states.
  <Expandable title='properties'>
    <ResponseField name='pending' type='object'>
      Mapping of sender addresses to nonce-indexed pending transactions.
    </ResponseField>
    <ResponseField name='queued' type='object'>
      Mapping of sender addresses to nonce-indexed queued transactions.
    </ResponseField>
  </Expandable>
</ResponseField>
