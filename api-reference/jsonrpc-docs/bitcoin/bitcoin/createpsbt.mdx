### Parameters

Parameters: [inputs, outputs, locktime?, replaceable?].

- `params[0]` (array): Transaction inputs (each with txid, vout, sequence).
- `params[1]` (object): Transaction outputs mapping addresses to amounts or scripts.
- `params[2]` (integer): Optional locktime.
- `params[3]` (boolean): Optional replaceable flag.

### Returns

<ResponseField name='result' type='string'>
  Newly crafted PSBT encoded as base64.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "createpsbt",
    "params": [
      [],
      {},
      0,
      false
    ]
  }'
```

</RequestExample>
