### Parameters

Parameters: [psbts].

- `params[]` (array): List of PSBT strings to merge.

### Returns

<ResponseField body='result' type='string'>
  Combined PSBT encoded as base64.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "combinepsbt",
    "params": [
      []
    ]
  }'
```

```json JSON-RPC

  ]
}
```
