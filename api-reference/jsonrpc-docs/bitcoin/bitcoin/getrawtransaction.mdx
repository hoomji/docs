---
title: 'getrawtransaction'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'none'
---
### Parameters

Parameters: [txid, verbose?].

- `params[0]` (string): 32 Bytes - hash of the transaction.
- `params[1]` (boolean): If true, return decoded JSON instead of hex.

### Returns

<ResponseField name='result' type='oneOf'>
  Raw transaction hex when verbose=false.
  <Expandable title='variants'>
    <ResponseField name='variant1' type='string'>
      Raw transaction hex when verbose=false.
    </ResponseField>
    <ResponseField name='variant2' type='object'>
      Decoded transaction JSON when verbose=true.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getrawtransaction",
    "params": [
      "",
      false
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": ""
}
```

</ResponseExample>
