### Parameters

Parameters: [txid, dummy, fee_delta].

- `params[0]` (string): 32 Bytes - hash of the transaction.
- `params[1]` (number): Unused parameter kept for compatibility (set to 0).
- `params[2]` (integer): Fee delta in satoshis added to virtual priority.

### Returns

<ResponseField body='result' type='boolean'>
  True if the transaction priority was adjusted. Example: <code>true</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "prioritisetransaction",
    "params": [
      "",
      0,
      0
    ]
  }'
```
