### Parameters

Parameters: [rawTx, privKeys, prevTxs?, sighashType?].

- `params[0]` (string): Raw transaction hex.
- `params[1]` (array): Private keys in WIF format.
- `params[2]` (array): Optional previous transaction outputs used for signing.
- `params[3]` (string): Optional signature hash type (e.g., ALL).

### Returns

<ResponseField name='result' type='object'>
  Signed transaction result containing hex and completion flag.
</ResponseField>
