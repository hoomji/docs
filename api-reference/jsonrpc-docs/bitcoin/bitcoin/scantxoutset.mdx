### Parameters

Parameters: [action, scanObjects].

- `params[0]` (string): Action (start, abort, status).
- `params[1]` (array): Descriptors or addresses to search for.

### Returns

<ResponseField body='result' type='object'>
  Scan status or results when complete.
</ResponseField>

  --header 'content-type: application/json' \
  --data '{
