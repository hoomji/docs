### Parameters

Parameters: [confTarget, estimateMode?].

- `params[0]` (integer): Target confirmation window in blocks.
- `params[1]` (string): Optional estimate mode (UNSET, ECONOMICAL, CONSERVATIVE).

### Returns

<ResponseField name='result' type='object'>
  Fee rate estimates in BTC/kvB along with error details.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "estimatesmartfee",
    "params": [
      0,
      ""
    ]
  }'
```

</RequestExample>

  "jsonrpc": "2.0",
