### Parameters

Parameters: [inputs, outputs, locktime?, replaceable?].

- `params[0]` (array): Array of inputs with txid and vout.
- `params[1]` (object): Outputs with address/script => amount mapping.
- `params[2]` (integer): Optional locktime.
- `params[3]` (boolean): Optional replaceable flag.

### Returns

<ResponseField body='result' type='string'>
  Raw transaction hex string.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "createrawtransaction",
    "params": [
      [],
      {},
      0,
      false
    ]
  }'
```
