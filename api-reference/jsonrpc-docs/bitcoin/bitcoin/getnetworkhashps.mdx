### Parameters

Parameters: [nblocks?, height?].

- `params[0]` (integer): Number of blocks to average over.
- `params[1]` (integer): Block height to estimate at (default best chain tip).

### Returns

<ResponseField name='result' type='number'>
  Estimated hashes per second.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getnetworkhashps",
    "params": [
      0,
      0
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": 0
}
```

</ResponseExample>
