title: 'logging'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [include?, exclude?].

- `params[]` (array): List of log categories to include or exclude.

### Returns

<ResponseField body='result' type='object'>
  Map of categories to boolean enabled state.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "logging",
    "params": [
      []
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "logging",
  "params": [
    []
  ]
}
```

</RequestExample>
