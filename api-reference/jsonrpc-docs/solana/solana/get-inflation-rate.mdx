---
title: 'getInflationRate'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'interactive'
---
### Parameters

This method does not require any parameters.

### Returns

<ResponseField name='result' type='object'>
  Breakdown of inflation allocation for a specific epoch.
  <Expandable title='properties'>
    <ResponseField name='total' type='number'>
      Total inflation rate (as a decimal).
    </ResponseField>
    <ResponseField name='validator' type='number'>
      Portion of inflation allocated to validators.
    </ResponseField>
    <ResponseField name='foundation' type='number'>
      Portion of inflation allocated to the foundation.
    </ResponseField>
    <ResponseField name='epoch' type='integer'>
      Epoch for which the inflation values apply.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getInflationRate",
    "params": []
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "total": 0.149,
    "validator": 0.148,
    "foundation": 0.001,
    "epoch": 100
  }
}
```

</ResponseExample>
