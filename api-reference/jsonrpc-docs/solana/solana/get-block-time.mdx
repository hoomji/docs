### Parameters

The parameters for method getBlockTime: [slot].

- `params[]` (integer): Slot number of the block to retrieve.

### Returns

<ResponseField body='result' type='integer'>
  Estimated production time of the block Example: <code>1574721591</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getBlockTime",
    "params": [
      5
    ]
  }'
```
