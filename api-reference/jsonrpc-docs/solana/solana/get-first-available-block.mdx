---
title: 'getFirstAvailableBlock'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

This method does not require any parameters.

### Returns

<ResponseField body='result' type='integer'>
  Slot of the lowest confirmed block that has not been purged from the ledger Example: <code>12345678</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getFirstAvailableBlock",
    "params": []
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "getFirstAvailableBlock",
  "params": []
}
```

</RequestExample>
