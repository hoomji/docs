### Parameters

Parameters for method getSupply: [config].

- `params[0]` (object): Optional configuration object.

### Returns

<ResponseField name='result' type='object'>
  Current supply information object.
  <Expandable title='properties'>
    <ResponseField name='total' type='integer'>
      Total supply in lamports.
    </ResponseField>
    <ResponseField name='circulating' type='integer'>
      Circulating supply in lamports.
    </ResponseField>
    <ResponseField name='nonCirculating' type='integer'>
      Non-circulating supply in lamports.
    </ResponseField>
    <ResponseField name='nonCirculatingAccounts' type='array'>
      Array of account addresses (base-58 encoded) that are non-circulating. Will be empty if excludeNonCirculatingAccountsList is true.
      <Expandable title='items'>
        <ResponseField name='items[]' type='string'>
          Base-58 encoded account address.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
