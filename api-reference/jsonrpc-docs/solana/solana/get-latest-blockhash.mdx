### Parameters

Parameters for method getLatestBlockhash: [config]

- `params[]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='object'>
  RPC response object containing blockhash info in the value field.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getLatestBlockhash",
    "params": [
      {
        "commitment": "processed"
      }
    ]
  }'
```
