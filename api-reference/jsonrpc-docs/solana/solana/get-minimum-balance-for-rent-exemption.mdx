### Parameters

Parameters for method getMinimumBalanceForRentExemption: [dataLength, config]

- `params[0]` (integer): Length of the data in bytes for which to calculate the minimum balance.
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField name='result' type='integer'>
  Minimum balance required to make an account rent exempt for the specified data length. Example: <code>203928</code>.
</ResponseField>
