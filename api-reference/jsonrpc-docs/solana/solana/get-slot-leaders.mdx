### Parameters

Parameters for method getSlotLeaders: [startSlot, limit]

- `params[0]` (integer): Starting slot number for the range.
- `params[1]` (integer): Maximum number of results to return.

### Returns

<ResponseField body='result' type='array'>
  Array of base-58 encoded public keys of slot leaders.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getSlotLeaders",
    "params": [
      100,
      10
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,

```
