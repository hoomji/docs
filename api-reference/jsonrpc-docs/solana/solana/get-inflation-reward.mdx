### Parameters

Parameters for method getInflationReward: [addresses, config]

- `params[0]` (array): List of addresses to get inflation rewards for.
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='array'>
  Array of reward objects for epochs in which rewards were credited.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getInflationReward",
    "params": [
      [
        "6dmNQ5jwLeLk5REvio1JcMshcbvkYMwy26sJ8pbkvStu",
        "BGsqMegLpV6n6Ve146sSX2dTjUMj3M92HnU8BbNRMhF2"
      ],
      {
        "epoch": 2
      }
    ]
  }'
```
