### Parameters

Parameters for method getTokenLargestAccounts: [mintAddress, config]

- `params[0]` (string): Pubkey of the token Mint to query (base-58 encoded string).
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='array'>
  Array of the 20 largest token accounts for the specified mint.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getTokenLargestAccounts",
    "params": [
      "3wyAj7Rt1TWVPZVteFJPLa26JmLvdb1CAKEFZm3NY75E"
    ]
  }'
```
