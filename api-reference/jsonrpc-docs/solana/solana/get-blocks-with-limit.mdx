### Parameters

Parameters for method getBlocksWithLimit: [startSlot, limit, config]

- `params[0]` (integer): Starting slot number for the range.
- `params[1]` (integer): Maximum number of results to return.
- `params[2]` (object): Optional configuration object.

### Returns

<ResponseField name='result' type='array'>
  Returns a list of confirmed blocks with a limit
</ResponseField>
