### Parameters

Parameters: [signedTransaction, config]

- `params[0]` (string): Fully-signed Transaction as an encoded string (base58 or base64 depending on config).
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='string'>
  First Transaction Signature embedded in the transaction (transaction ID), base-58 encoded string. Example: <code>"5np2V7ZyrjULExB7T9MfU2v6KH3eEAYb5hNDqgCVz3oF"</code>.
</ResponseField>
