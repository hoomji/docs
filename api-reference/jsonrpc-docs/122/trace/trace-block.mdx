### Parameters

Parameters for trace_block method: [defaultBlock]

- `params[0]` (integer): Integer block number.
- `params[1]` (string): Block tag: "earliest", "latest", or "pending".

### Returns

<ResponseField body='result' type='array'>
  Array - Block traces.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=122 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "trace_block",
    "params": [
      "0x2ed119"
    ]
  }'
```

