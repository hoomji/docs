### Parameters

Parameters for method trace_replayTransaction: [hash, traceTypes].

- `params[0]` (string): 32 Bytes - Hash of the transaction.
- `params[1]` (array): Types of traces to collect: "vmTrace", "trace", and/or "stateDiff".

### Returns

<ResponseField name='result' type='object'>
  Object - Block traces.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=994873017 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "trace_replayTransaction",
    "params": [
      "0x02d4a872e096445e80d05276ee756cefef7f3b376bcec14246469c0cd97dad8f",
      [
        "trace"
      ]
    ]
  }'
```
