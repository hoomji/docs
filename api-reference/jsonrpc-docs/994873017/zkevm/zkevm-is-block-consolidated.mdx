title: 'zkevm_isBlockConsolidated'

playground: 'none'

### Parameters

- `params[]` (string): The block number to check if it is connected to a verified batch, encoded as a hexadecimal string.

### Returns

<ResponseField name='result' type='boolean'>
  True if the block is connected to a verified batch (consolidated), false otherwise. Example: <code>true</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=994873017 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "zkevm_isBlockConsolidated",
    "params": [
      "0x2a3b"
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": true
}
```

</ResponseExample>
