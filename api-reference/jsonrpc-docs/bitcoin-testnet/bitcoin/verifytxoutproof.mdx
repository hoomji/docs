---
title: 'verifytxoutproof'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [proof, txids?].

- `params[0]` (string): Proof data hex.
- `params[1]` (array): Optional list of transaction ids expected in the proof.

### Returns

<ResponseField body='result' type='array'>
  Transaction ids verified by the proof.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "verifytxoutproof",
    "params": [
      "",
      []
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "verifytxoutproof",

    []
  ]
}
```
