---
title: 'zkevm_batchNumberByBlockNumber'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'none'
---
### Parameters

- `params[]` (string): The block number to get the batch number for, encoded as a hexadecimal string.

### Returns

<ResponseField name='result' type='string'>
  The batch number of the specified block number in the zkEVM chain, returned as a hexadecimal string. Example: <code>"0x1a"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1101 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "zkevm_batchNumberByBlockNumber",
    "params": [
      "0x2a3b"
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1a"
}
```

</ResponseExample>
