### Parameters

Two-element array: [modified transaction object, default block]. The transaction omits "nonce" and allows optional "from".

- `params[0]` (allOf): Legacy transaction (pre-EIP-2718).
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField body='result' type='string'>
  The return value of the executed contract. Example: <code>"0x"</code>.
</ResponseField>
