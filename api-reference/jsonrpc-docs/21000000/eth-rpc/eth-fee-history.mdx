title: 'eth_feeHistory'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [blockCount, newestBlock, rewardPercentiles]

- `params[0]` (string): Hex-encoded number of blocks in the requested range (max: 0x400 / 1024 blocks).
- `params[1]` (string): Newest block identifier: a hex block number or one of: "latest", "earliest", "pending".
- `params[2]` (array): Optional array of integers representing percentiles (ascending).

### Returns

<ResponseField body='result' type='object'>
  Historical gas fee statistics over the requested range of blocks.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=21000000 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_feeHistory",
    "params": [
      "0x5",
      "latest",
      [
        20,
        30
      ]
    ]
  }'
```
