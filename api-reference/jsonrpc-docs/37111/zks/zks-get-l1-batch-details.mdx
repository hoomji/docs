### Parameters

- `params[]` (integer): The layer 1 batch number to query details for.

### Returns

<ResponseField body='result' type='object'>
  The JSON-RPC result.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=37111 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "zks_getL1BatchDetails",
    "params": [
      12345
    ]
  }'
```
