### Parameters

- `params[0]` (integer): Block height for gas price query
- `params[1]` (string): Block hash for gas price query
- `params[2]` (string): Fetch gas price for the latest block

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='gas_price' type='string'>
      The gas price in yoctoNEAR
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=398 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "gas_price",
    "params": [
      0,
      "",
      ""
    ]
  }'
```

</RequestExample>

  "jsonrpc": "2.0",
