### Parameters

Specify either `finality` or `block_id` to get protocol config at a given block state.

- `finality` (string): Optional finality level to query protocol config at. Cannot be used with block_id.
- `block_id` (oneOf): Block height or hash. Cannot be used with finality.

  --header 'content-type: application/json' \

    "id": 1,
    "jsonrpc": "2.0",
    "method": "EXPERIMENTAL_protocol_config",

    }
