### Parameters

Checks status of a transaction by hash and includes all receipt outcomes and metadata.

- `tx_hash` (string): Base58-encoded transaction hash.
- `sender_account_id` (string): Account ID of the sender. Used to determine which shard to search in.
- `wait_until` (string): Optional execution level to wait for before responding. Use NONE for immediate response.

### Returns

<ResponseField body='result' type='object'>
  Complete transaction status, including all related receipts and outcomes.
</ResponseField>
