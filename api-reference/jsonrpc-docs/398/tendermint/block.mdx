### Parameters

The parameters for method block: [height].

- `params[]` (integer): Integer - height to return. If no height is provided, it will fetch the latest block.

### Returns

<ResponseField name='result' type='object'>
  Block information.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=398 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "block",
    "params": [
      1
    ]
  }'
```

</RequestExample>
