---
title: 'arbtrace_block'
api: 'POST https://api.uniblock.dev/uni/v1/json-rpc'
playground: 'none'
---
### Parameters

Array containing a single block number (as hex) or tag (e.g., "latest").

- `params[]` (string): The block number (hex) or tag ("latest", "pending").

### Returns

<ResponseField name='result' type='array'>
  List of trace objects for all transactions in the block.

    <ResponseField name='items[]' type='object'>
