### Parameters

Parameters for method debug_traceBlockByHash: [blockHash, traceConfig]

- `params[0]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.
- `params[1]` (object): Optional configuration object passed to tracing RPC methods.

### Returns

<ResponseField name='result' type='array'>
  Full stack trace.
</ResponseField>
