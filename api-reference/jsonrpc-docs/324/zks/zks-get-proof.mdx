  <ParamField query='chainId' type='string' default='324' required>

<ParamField body='params' type='object' default={["0x0000000000000000000000000000000000008003",["0x8b65c0cf1012ea9f393197eb24619fd814379b298b238285649e14f936a5eb12"],354895]} placeholder='["0x0000000000000000000000000000000000008003",["0x8b65c0cf1012ea9f393197eb24619fd814379b298b238285649e14f936a5eb12"],354895]' required>
  Positional parameters for <code>zks_getProof</code>:
  <ul>
  <li><code>params[0]</code>: The account address to fetch storage proofs for.</li>
  <li><code>params[1]</code>: Vector of storage keys in the account.</li>
  <li><code>params[2]</code>: The L1 batch number specifying the point in time for the storage proof.</li>
  </ul>
</ParamField>
