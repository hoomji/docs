---
title: 'zks_getRawBlockTransactions'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Block number to get raw transaction data from.

- `params[]` (integer): The L2 block number.

### Returns

<ResponseField body='result' type='array'>
  List of executed transactions in the given block.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=324 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "zks_getRawBlockTransactions",
    "params": [
      5817
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "zks_getRawBlockTransactions",
  "params": [
    5817
  ]
}
```

</RequestExample>
