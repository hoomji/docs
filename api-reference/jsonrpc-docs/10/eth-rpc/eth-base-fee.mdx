### Parameters

This method does not accept any parameters.

### Returns

<ResponseField body='result' type='string'>
  Hex-encoded base fee for the next block (in wei). Example: <code>"0x2e90edd000"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=10 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_baseFee",
    "params": [
      null
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_baseFee",
  "params": [
    null
  ]
}
```

</RequestExample>
