### Parameters

This method does not require any parameters.

### Returns

<ResponseField body='result' type='string'>
  20 Bytes - the current coinbase address. Example: <code>"0x407d73d8a49eeb85d32cf465507dd71d507100c1"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=2020 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_coinbase",
    "params": []
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_coinbase",
  "params": []
}
```

</RequestExample>
