---
title: 'eth_getTransactionReceiptsByBlock'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [blockReference]

- `params[]` (string): The block number in hex, block hash, or a tag such as "latest", "earliest", or "finalized".

### Returns

<ResponseField body='result' type='array'>
  An array of transaction receipt objects or null if no receipts were found.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1088 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getTransactionReceiptsByBlock",
    "params": [
      "latest"
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getTransactionReceiptsByBlock",
  "params": [
    "latest"
  ]
}
```

</RequestExample>
