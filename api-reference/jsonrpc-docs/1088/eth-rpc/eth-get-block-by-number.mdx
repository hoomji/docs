### Parameters

  <ParamField query='chainId' type='string' default='1088' required>
    Unique identifier for a blockchain network.
  </ParamField>
  <ParamField body='id' type='number' default={1} required>
    Request identifier used to match responses.
  </ParamField>
  <ParamField body='jsonrpc' type='string' default='2.0' required>
    JSON-RPC version.
  </ParamField>
  <ParamField body='method' type='string' default='eth_getBlockByNumber' required>
    JSON-RPC method name.
  </ParamField>

<ParamField body='params' type='object' default={["0x1b4",true]} placeholder='["0x1b4",true]' required>
  Parameters for eth_getBlockByNumber: [blockNumber, fullTransactions].
  
  Positional parameters for <code>eth_getBlockByNumber</code>:
  <ul>
  <li><code>params[0]</code>: Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".</li>
  <li><code>params[1]</code>: If true, returns the full transaction objects. If false, returns only the transaction hashes.</li>
  </ul>
</ParamField>
