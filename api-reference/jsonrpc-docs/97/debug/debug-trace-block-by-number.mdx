### Parameters

Parameters for method debug_traceBlockByNumber: [blockNumber, traceConfig]

- `params[0]` (oneOf): Block number where this transaction was included. Null when pending.
- `params[1]` (object): Optional configuration object passed to tracing RPC methods.

### Returns

<ResponseField body='result' type='array'>
  Full stack trace.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=97 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "debug_traceBlockByNumber",
    "params": [
      "0xccde12",
      {
        "tracer": "callTracer"
      }
    ]
  }'
```
