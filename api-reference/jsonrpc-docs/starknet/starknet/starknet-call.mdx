### Parameters

Parameters for starknet_call: [contract_address, entry_point_selector, calldata].

- `request` (object): Function call object.
- `block_id` (object): BlockID object containing hash and number.

### Returns

<ResponseField body='result' type='array'>
  The function return values as an array of strings.
</ResponseField>
