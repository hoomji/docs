### Parameters

Parameters for method starknet_getClassHashAt: [block_id, contract_address]

- `block_id` (object): BlockID object containing hash and number.
- `contract_address` (string): Address of the requested contract class.

### Returns

<ResponseField body='result' type='string'>
  The class hash of the requested contract class (DATA, 32 Bytes). Example: <code>"0xabcdefabcdefabcdefabcdefabcdefabcdefabcdefabcdefabcdefabcdefabcdef"</code>.
</ResponseField>
