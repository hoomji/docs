### Parameters

Paramters for method starknet_getTransactionReceipt: [transaction_hash].

- `transaction_hash` (string): 32 Bytes - Hash of the transaction.

### Returns

<ResponseField body='result' type='object'>
  The transaction receipt object
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=starknet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "starknet_getTransactionReceipt",
    "params": {
      "transaction_hash": "0x121aaba2894a63cce81acabb559eb2c114f4859dc5803e0b91349ec398d2eb0"
    }
  }'
```
