### Parameters

The parameters for method tx: [transactionHash, prove].

- `params[0]` (string): 32 Bytes - Hash of the transaction.
- `params[1]` (boolean): To include proofs into the response.

### Returns

<ResponseField body='result' type='object'>
  The transactions with or without proof information.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1329 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "tx",
    "params": [
      "0xD70952032620CC4E2737EB8AC379806359D8E0B17B0488F627997A0B043ABDED",
      true
    ]
  }'
```
