### Parameters

- `params[0]` (object): The query object to filter specific events

### Returns

<ResponseField body='result' type='object'>
  Real-time event stream payloads from the Tendermint network
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1329 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "events",
    "params": [
      {}
    ]
  }'
```
