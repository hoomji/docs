### Parameters

This method does not require any parameters.

### Returns

<ResponseField body='result' type='object'>
  The genesis information.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1329 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "genesis",
    "params": []
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "genesis",
  "params": []
}
```
