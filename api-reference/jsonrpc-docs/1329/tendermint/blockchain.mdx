### Parameters

The parameters for method blockchain: [minHeight, maxHeight].

- `params[0]` (integer): Integer - minHeight to return.
- `params[1]` (integer): Integer - maxHeight to return.

### Returns

<ResponseField name='result' type='object'>
  The block headers.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1329 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "blockchain",
    "params": [
      1,
      100
    ]
  }'
```

</RequestExample>
