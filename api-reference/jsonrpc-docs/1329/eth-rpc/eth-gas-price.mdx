### Returns

<ResponseField name='result' type='string'>

</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_gasPrice",
    "params": []
  }'
```

</RequestExample>
