### Parameters

Parameters for eth_newFilter: [filterObject]. See below for filterObject properties.

- `params[0]` (object): Filter options used to create a new filter for logs. Topics are order-dependent. Nulls or nested arrays are used to match multiple possibilities.

### Returns

<ResponseField name='result' type='string'>
  Quantity - The filter ID. Example: <code>"0x1"</code>.
</ResponseField>
