### Parameters

Request parameters for retrieving asset proof.

- `id` (string): The ID of the compressed asset.

### Returns

<ResponseField body='result' type='object'>
  Merkle proof data for the given asset.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana-devnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getAssetProof",
    "params": {
      "id": "Bu1DEKeawy7txbnCEJE4BU3BKLXaNAKCYcHR4XhndGss"
    }
  }'
```

