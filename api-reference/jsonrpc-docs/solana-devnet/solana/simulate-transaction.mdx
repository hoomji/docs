### Parameters

Parameters for method simulateTransaction: [transaction, config]

- `params[0]` (string): Transaction as an encoded string (base58 or base64 depending on config). The transaction must have a valid blockhash but is not required to be signed.
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='object'>
  Simulated transaction execution result.
</ResponseField>
