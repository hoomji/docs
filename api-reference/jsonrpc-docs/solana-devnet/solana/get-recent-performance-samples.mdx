### Parameters

The parameters for method getRecentPerformanceSamples: [samples].

- `params[]` (integer): Number of samples to return.

### Returns

<ResponseField body='result' type='array'>
  Array of performance sample objects representing validator activity during sample periods.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana-devnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getRecentPerformanceSamples",
    "params": [
      5
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "getRecentPerformanceSamples",
  "params": [
    5
  ]
}
```

</RequestExample>
