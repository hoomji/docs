### Parameters

Parameters for method getBalance: [pubkey, config]

- `params[0]` (string): Base-58 encoded public key of the account to query.
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='object'>
  Lamport balance of the account of provided Pubkey
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana-devnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getBalance",
    "params": [
      "83astBRguLMdt2h5U1Tpdq5tjFoJ6noeGwaY3mDLVcri"
    ]
  }'
```
