### Parameters

Parameters for method getBlocks: [startSlot, endSlot, config]

- `params[0]` (integer): Starting slot number for the range.
- `params[1]` (integer): Ending slot number for the range.
- `params[2]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='array'>
  Returns a list of confirmed blocks with a limit
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana-devnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getBlocks",
    "params": [
      5,
      10
    ]
  }'
```
