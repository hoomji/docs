### Parameters

Parameters for method getSignatureStatuses: [signatures, config]

- `params[0]` (array): Array of transaction signatures (txids), base-58 encoded (up to 256).
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='array'>
  Array of signature status responses.
</ResponseField>
