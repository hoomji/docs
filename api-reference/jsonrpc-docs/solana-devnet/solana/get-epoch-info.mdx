### Parameters

Parameters for method getEpochInfo: [config]

- `params[]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='object'>
  Information about the current slot, epoch, and transaction count.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana-devnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getEpochInfo",
    "params": [
      {}
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "getEpochInfo",
  "params": [
    {}
  ]
}
```

</RequestExample>
