### Parameters

Parameters for method getTransactionCount: [config]

- `params[]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='integer'>
  The current Transaction count from the ledger. Example: <code>268</code>.
</ResponseField>

```bash Curl

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getTransactionCount",
    "params": [
      {}
    ]
  }'
```
