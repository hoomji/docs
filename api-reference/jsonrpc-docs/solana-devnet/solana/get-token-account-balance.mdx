### Parameters

Parameters for method getTokenAccountBalance: [tokenAccountAddress, config]

- `params[0]` (string): Pubkey of the Token account to query (base-58 encoded string).
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='object'>
  Token balance information for the given Token account.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=solana-devnet \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getTokenAccountBalance",
    "params": [
      "7fUAJdStEuGbc3sM84cKRL6yYaaSstyLSU4ve5oovLS7"
    ]
  }'
```
