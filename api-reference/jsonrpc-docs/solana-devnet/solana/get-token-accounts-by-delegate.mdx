### Parameters

Parameters for method getTokenAccountsByDelegate: [delegateAddress, filter, config]

- `params[0]` (string): Pubkey of the account delegate (base-58 encoded).
- `params[1]` (object): Filter object specifying either mint or programId (exactly one required).
- `params[2]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='array'>
  Array of token accounts owned by delegate.
</ResponseField>
