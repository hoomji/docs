### Parameters

- `changes_type` (string): Must be `"single_access_key_changes"`
- `keys` (array): List of (account_id, public_key) pairs to query
- `finality` (string): Either `finality` or `block_id`, not both
- `block_id` (oneOf): Either `block_id` or `finality`, not both

### Returns

<ResponseField body='result' type='object'>
  The JSON-RPC result.
</ResponseField>
