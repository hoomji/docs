### Parameters

Submits a signed transaction and optionally waits for a specific level of execution confirmation.

- `signed_tx_base64` (string): Base64-encoded signed transaction.
- `wait_until` (string): Minimal execution level required. Determines when the response is returned.

### Returns

<ResponseField body='result' type='object'>
  Complete status and outcome details of the submitted transaction.
</ResponseField>
