### Parameters

Parameters for eth_submitHashrate: [hashrate, id].

- `params[0]` (string): Data - a hexadecimal string representation (32 bytes) of the hash rate.
- `params[1]` (string): Data - A random hexadecimal(32 bytes) ID identifying the client.

### Returns

<ResponseField body='result' type='boolean'>
  Boolean - Returns true if submitting hash rate was successful. Example: <code>true</code>.
</ResponseField>
