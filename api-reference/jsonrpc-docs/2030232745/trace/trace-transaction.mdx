### Parameters

Parameters for method trace_transaction: [transactionHash].

- `params[]` (string): 32 Bytes - Hash of the transaction.

### Returns

<ResponseField body='result' type='array'>
  Array - Traces of given transaction
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=2030232745 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "trace_transaction",
    "params": [
      "0x17104ac9d3312d8c136b7f44d4b8b47852618065ebfa534bd2d3b5ef218ca1f3"
    ]
  }'
```
