---
title: 'zkevm_isBlockVirtualized'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

- `params[]` (string): The block number to check if it has been virtualized, encoded in hexadecimal format.

### Returns

<ResponseField body='result' type='boolean'>
  True if the specified block has been virtualized, otherwise false. Example: <code>true</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=2030232745 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "zkevm_isBlockVirtualized",
    "params": [
      "0x245f"
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "zkevm_isBlockVirtualized",
  "params": [
    "0x245f"
  ]
}
```

</RequestExample>
