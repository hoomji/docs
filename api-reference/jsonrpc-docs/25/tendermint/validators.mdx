### Parameters

The parameters for method validators: [height, page, perPage].

- `params[0]` (integer): The height to return.
- `params[1]` (integer): Page number.
- `params[2]` (integer): Number of items per page.

### Returns

<ResponseField body='result' type='object'>
  The validator information.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=25 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "validators",
    "params": [
      0,
      0,
      0
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "validators",
  "params": [
    0,
    0,
    0
  ]
}
```

</RequestExample>
