### Parameters

Query parameters for method abci_query in the order: [path, data, height, prove]

- `params[0]` (string): Path to the data (e.g., "/a/b/c")
- `params[1]` (string): Data content to query
- `params[2]` (integer): Block height. 0 means latest.
- `params[3]` (boolean): Whether to include proofs of transaction inclusion.

### Returns

<ResponseField body='result' type='object'>
  Application query response.
</ResponseField>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=25 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "abci_query",
    "params": [

      1,

    ]
  }'
```
