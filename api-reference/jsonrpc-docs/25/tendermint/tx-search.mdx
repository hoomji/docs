### Parameters

Transaction search parameters in the order: [query, prove, page, per_page, order_by, match_events]

- `params[0]` (string): A transaction query string (e.g., "tx.height=1000").
- `params[1]` (boolean): Include proofs of the transaction inclusion in the block.
- `params[2]` (integer): Page number (1-based).
- `params[3]` (integer): Number of entries per page (max: 100).
- `params[4]` (string): Order of transaction sorting ("asc" or "desc"), by height & index.
- `params[5]` (boolean): Match attributes in query within events in addition to height & txhash.

### Returns

<ResponseField name='result' type='object'>
  The transactions returned with the search queries.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=25 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "tx_search",
    "params": [
      "tx.height=1000",
      true,
      1,
      30,
      "asc",
      true
    ]
  }'
```
