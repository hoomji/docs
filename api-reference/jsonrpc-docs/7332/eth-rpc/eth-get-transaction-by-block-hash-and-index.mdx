### Parameters

Parameters for eth_getTransactionByBlockHashAndIndex: [blockHash, index].

- `params[0]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.
- `params[1]` (integer): Integer of the transaction index position.
