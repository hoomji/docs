---
title: 'klay_getBlockByNumber'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [blockNumber, fullTransactions].

- `params[0]` (oneOf): Block number where this transaction was included. Null when pending.
- `params[1]` (boolean): If true, returns full transaction objects instead of hashes.

### Returns

<ResponseField body='result' type='object'>
  A Klaytn block object. Null if the block cannot be found.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1001 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "klay_getBlockByNumber",
    "params": [
      "0x1b4",
      false
    ]
  }'
```
