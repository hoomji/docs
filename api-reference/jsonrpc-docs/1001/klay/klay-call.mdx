### Parameters

Parameters: [call object, block parameter]. The transaction object omits "nonce" and may omit "from".

- `params[0]` (allOf): Legacy transaction (pre-EIP-2718).
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField name='result' type='string'>
  Return data from the executed call as a hex string. Example: <code>"0x"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1001 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "klay_call",
    "params": [
      {
        "from": "0x1b3c8c1cdb4e6b6b8819345bf3c6d9c926f55f90",
        "to": "0xa94f5374fce5edbc8e2a8697c15331677e6ebf0b",
        "data": "0x"
      },
      "latest"
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success

  "jsonrpc": "2.0",
```

</ResponseExample>
