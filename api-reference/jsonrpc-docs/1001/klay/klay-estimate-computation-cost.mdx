### Parameters

Parameters: [transaction object, block parameter].

- `params[0]` (object): Legacy transaction (pre-EIP-2718).
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField name='result' type='object'>
  Estimated computation cost details.
  <Expandable title='properties'>
    <ResponseField name='gas' type='string'>
      Estimated gas limit (hex).
    </ResponseField>
    <ResponseField name='gasPrice' type='string'>
      Suggested gas price (hex).
    </ResponseField>
    <ResponseField name='feeRatio' type='string'>
      Optional fee ratio applied to the transaction.
    </ResponseField>
  </Expandable>
</ResponseField>
