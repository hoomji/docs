### Parameters

Parameters: [address, block parameter].

- `params[0]` (string): The address.
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField body='result' type='object'>
  Account metadata at the specified block.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1001 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_getAccount",
    "params": [
      "",
      0
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "kaia_getAccount",
  "params": [
    "",
    0
  ]
}
```

</RequestExample>
