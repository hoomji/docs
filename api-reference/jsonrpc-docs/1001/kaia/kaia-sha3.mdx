### Parameters

Parameters: [data].

- `params[]` (string): Data - (optional) 4 byte hash of the method signature followed by encoded parameters.

### Returns

<ResponseField name='result' type='string'>
  Keccak-256 hash of the input data. Example: <code>"0x47173285a8d7341e5e972fc677286384f802f8ef42a5ec5f03bbfa254cb01fad"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1001 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_sha3",
    "params": [
      "0x68656c6c6f20776f726c64"
    ]
  }'
```

</RequestExample>
