### Parameters

Parameters: [call object, block parameter]. Call object follows Kaia transaction fields with nonce omitted.

- `params[0]` (allOf): Legacy transaction (pre-EIP-2718).
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField name='result' type='string'>
  The return data from executing the call, encoded as a hex string. Example: <code>"0x"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1001 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_call",
    "params": [
      {
        "from": "0x8d97689c9818892b700e27f316cc3e41e17fbeb9",
        "to": "0xf20d962a6c8f70c731bd838a3a388d7d48fa6e15",
        "data": "0x"
      },
      "latest"
    ]
  }'
```

</RequestExample>
