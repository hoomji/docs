### Parameters

Parameters: [blockNumber, transactionIndex].

- `params[0]` (oneOf): Block number where this transaction was included. Null when pending.
- `params[1]` (integer): The transaction's index position in the block. Null when pending.

### Returns

<ResponseField body='result' type='string'>
  The transaction.
</ResponseField>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=1001 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_getTransactionByBlockNumberAndIndex",
    "params": [

      0
    ]
  }'
```
