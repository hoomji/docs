### Parameters

Parameters: [address, block parameter]. Block parameter defaults to "latest" when omitted.

- `params[0]` (string): The address.
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField name='result' type='boolean'>
  True if the account has been created, otherwise false. Example: <code>true</code>.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "kaia_accountCreated",
    "params": [
      "0x8d97689c9818892b700e27f316cc3e41e17fbeb9",
      "latest"
    ]
  }'
```
