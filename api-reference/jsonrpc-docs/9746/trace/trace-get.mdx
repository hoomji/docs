### Parameters

Parameters for method trace_get: [transactionHash, index].

- `params[0]` (string): 32 Bytes - Hash of the transaction.
- `params[1]` (array): Array - Index positions of the traces.

### Returns

<ResponseField body='result' type='object'>
  Object - Trace object
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=9746 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "trace_get",
    "params": [
      "0x17104ac9d3312d8c136b7f44d4b8b47852618065ebfa534bd2d3b5ef218ca1f3",
      [
        "0x0"
      ]
    ]
  }'
```
