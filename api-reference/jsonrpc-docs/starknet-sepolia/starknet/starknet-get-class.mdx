### Parameters

Parameters for starknet_getClass: [block_id, class_hash].

- `block_id` (object): BlockID object containing hash and number.
- `class_hash` (string): The class hash of the requested contract class.

### Returns

<ResponseField body='result' type='object'>
  Contract class object.
</ResponseField>

