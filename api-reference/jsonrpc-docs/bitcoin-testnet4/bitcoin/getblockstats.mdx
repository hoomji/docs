---
title: 'getblockstats'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [hash_or_height, metrics?].

- `params[0]` (string): Block hash or height.
- `params[1]` (array): Optional list of metrics to compute.

### Returns

<ResponseField body='result' type='object'>
  Selected block statistics (size, weight, fees, etc.).
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet4 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getblockstats",
    "params": [
      "",
      []
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "getblockstats",
  "params": [
    "",
    []
  ]
}
```

</RequestExample>
