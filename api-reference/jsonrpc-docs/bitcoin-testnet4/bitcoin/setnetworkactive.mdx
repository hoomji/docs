title: 'setnetworkactive'

### Parameters

Parameters: [state].

- `params[]` (boolean): True to enable networking, false to disable.

### Returns

<ResponseField name='result' type='boolean'>
  True if the networking state was updated. Example: <code>true</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet4 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "setnetworkactive",
    "params": [
      false
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": true
}
```

</ResponseExample>
