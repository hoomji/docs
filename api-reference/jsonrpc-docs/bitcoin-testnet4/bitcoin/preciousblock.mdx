### Parameters

Parameters: [blockHash].

- `params[]` (string): 32 Bytes - hash of the transaction.

### Returns

<ResponseField name='result' type='boolean'>
  True if the block was marked precious. Example: <code>true</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet4 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "preciousblock",
    "params": [
      ""
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": true
```
