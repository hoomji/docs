### Parameters

Parameters: [proof, txids?].

- `params[0]` (string): Proof data hex.
- `params[1]` (array): Optional list of transaction ids expected in the proof.

### Returns

<ResponseField name='result' type='array'>
  Transaction ids verified by the proof.
  <Expandable title='items'>
    <ResponseField name='items[]' type='string'>
      32 Bytes - hash of the transaction.
    </ResponseField>
  </Expandable>
</ResponseField>

  --header 'content-type: application/json' \
  --data '{
    "id": 1,

      "",
      []
