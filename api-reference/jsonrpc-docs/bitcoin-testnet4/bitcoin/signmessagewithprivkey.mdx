---
title: 'signmessagewithprivkey'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [privkey, message].

- `params[0]` (string): Private key in WIF.
- `params[1]` (string): Message to sign.

### Returns

<ResponseField body='result' type='string'>
  Base64-encoded signature.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet4 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "signmessagewithprivkey",
    "params": [
      "",
      ""
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "signmessagewithprivkey",

    "",
    ""
  ]
}
```

</RequestExample>
