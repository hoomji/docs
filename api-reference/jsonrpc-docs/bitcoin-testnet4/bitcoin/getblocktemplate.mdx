### Parameters

Parameters: [template_request?].

- `params[]` (object): Optional template request specifying capabilities and rules.

### Returns

<ResponseField name='result' type='object'>
  Block template data including transactions and coinbase details.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet4 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getblocktemplate",
    "params": [
      {}
    ]
  }'
```
