### Parameters

Parameters: [rawTransactions, maxfeerate?].

- `params[0]` (array): Array of raw transaction hex strings.
- `params[1]` (number): Optional max fee rate (BTC/kvB).

### Returns

<ResponseField body='result' type='array'>
  Package acceptance results per transaction.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet4 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "submitpackage",
    "params": [
      [],
      0
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "submitpackage",

    [],
    0
  ]
}
```
