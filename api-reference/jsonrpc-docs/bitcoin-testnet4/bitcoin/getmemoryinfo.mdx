title: 'getmemoryinfo'
openapi: 'POST /json-rpc'
playground: 'interactive'

### Parameters

Parameters: [mode?].

- `params[]` (string): Optional mode ("stats" or "mallocinfo").

### Returns

<ResponseField body='result' type='object'>
  Memory statistics or malloc info depending on mode.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=bitcoin-testnet4 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getmemoryinfo",
    "params": [
      ""
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "getmemoryinfo",
  "params": [
    ""
  ]
}
```

</RequestExample>
