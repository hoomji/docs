### Parameters

Parameters for method debug_traceCall: [transactionObject, blockNrOrHash, traceConfig].

- `params[0]` (object): Legacy transaction (pre-EIP-2718).
- `params[1]` (string): The block number or hash.
- `params[2]` (object): Optional configuration object passed to tracing RPC methods.

### Returns

<ResponseField body='result' type='object'>
  The tracer object.
</ResponseField>
