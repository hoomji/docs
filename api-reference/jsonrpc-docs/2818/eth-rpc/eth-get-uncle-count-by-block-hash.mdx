### Parameters

Parameters for eth_getUncleCountByBlockHash: [blockHash].

- `params[0]` (string): 32 Bytes - hash of the block where this transaction was included. Null when pending.

### Returns

<ResponseField body='result' type='string'>
  The number of uncles in the block. Example: <code>"0x1b"</code>.
</ResponseField>
