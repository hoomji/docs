### Parameters

Parameters for method getTransaction: [transactionSignature, config]

- `params[0]` (string): Transaction signature as base-58 encoded string.
- `params[1]` (object): Optional configuration object.

### Returns

<ResponseField body='result' type='oneOf'>
  Transaction response object if found and confirmed.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=stellar-soroban \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getTransaction",
    "params": [
      "2nBhEBYYvfaAe16UMNqRHre4YNSskvuYgx3M6E4JP1oDYvZEJHvoPzyUidNgNX5r9sTyN1J9UxtbCXy2rqYcuyuv",
      "json"
    ]
  }'
```
