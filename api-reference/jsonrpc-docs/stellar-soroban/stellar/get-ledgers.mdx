### Parameters

- `startLedger` (integer): Inclusive starting ledger sequence to list from.
- `order` (string): Sort order by ledger sequence.
- `pagination` (object): Pagination controls.

### Returns

<ResponseField name='result' type='object'>
  Horizon ledgers page.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=stellar-soroban \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "getLedgers",
    "params": {
      "startLedger": "LEDGER_SEQUENCE",
      "pagination": {
        "limit": 10
      }
    }
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {}
}
```

</ResponseExample>
