title: 'eth_getAssetBalance'

playground: 'none'

### Parameters

Parameters: [address, blk, assetID]

- `params[0]` (string): The address of the asset owner.
- `params[1]` (string): Block identifier: block number or block hash (hex or tag).
- `params[2]` (string): The asset ID for which the balance is requested.

### Returns

<ResponseField name='result' type='string'>
  Hexadecimal string representing the asset balance. Example: <code>"0x16345785d8a0000"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=14 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getAssetBalance",
    "params": [
      "0x8D97689C9818892B700e27F316cc3E41e17fBeb9",
      "latest",
      "0xf20d962a6c8f70c731bd838a3a388d7d48fa6e15"
    ]
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,

}
```

</ResponseExample>
