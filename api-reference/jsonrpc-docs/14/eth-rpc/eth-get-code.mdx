### Parameters

Parameters for eth_getCode: [address, defaultBlock].

- `params[0]` (string): The address.
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField body='result' type='string'>
  The code from the given address. Example: <code>"0x600160008035811a818181146012578301005b601b6001356025565b8060005260206000f35b600060078202905091905056"</code>.
</ResponseField>

curl --request POST \

  --header 'content-type: application/json' \
