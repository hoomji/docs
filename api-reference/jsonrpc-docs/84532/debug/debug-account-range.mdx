### Parameters

Parameters: [blockIdentifier, startKey, maxResults, context?].

- `params[0]` (string): Block number or hash at which to inspect state (hex quantity or 32-byte hash).
- `params[1]` (string): Hex-encoded start account key (inclusive).
- `params[2]` (integer): Maximum number of accounts to return.
- `params[3]` (object): Optional context returned from a previous call to continue iteration.

### Returns

<ResponseField body='result' type='object'>
  Account range response including account details and continuation token.
</ResponseField>
