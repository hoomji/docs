### Parameters

Parameters for the trace_replayBlockTransactions method: [defaultBlock, traceTypes].

- `params[0]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".
- `params[1]` (array): Types of traces to collect: "vmTrace", "trace", and/or "stateDiff".

### Returns

<ResponseField body='result' type='array'>
  Array - Block transactions traces.
</ResponseField>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=84532 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "trace_replayBlockTransactions",
```
