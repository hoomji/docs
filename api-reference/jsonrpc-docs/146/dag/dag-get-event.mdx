### Parameters

- `params[0]` (string): The full event ID (hex-encoded 32 bytes) or short event ID.

### Returns

<ResponseField body='result' type='object'>
  Returns an event object, or null if not found.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=146 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "dag_getEvent",
    "params": [
      "0x000170430000080cd8f7b782afe5d1f63be6cd73631598313200777062ba7383"
    ]
  }'
```
