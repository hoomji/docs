### Parameters

Parameters for eth_getBlockRange: [startBlockTag, endBlockTag, fullTxObjects]

- `params[0]` (string): Start block number (hex) or tag (latest, earliest, pending, safe, finalized).
- `params[1]` (string): End block number (hex) or tag (latest, earliest, pending, safe, finalized).
- `params[2]` (boolean): Set to true to return full transaction objects, false for only transaction hashes.

### Returns

<ResponseField body='result' type='array'>
  List of block objects in the specified range.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=5003 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getBlockRange",
    "params": [
      "0x1",
      "0x2",
      false
    ]
  }'
```
