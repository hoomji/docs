### Parameters

This method does not require any parameters.

### Returns

<ResponseField name='result' type='string'>
  20 Bytes - the current coinbase address. Example: <code>"0x407d73d8a49eeb85d32cf465507dd71d507100c1"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=5000 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_coinbase",
    "params": []
  }'
```

</RequestExample>
<ResponseExample>

```json Success
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x407d73d8a49eeb85d32cf465507dd71d507100c1"
}
```

</ResponseExample>
