### Parameters

The parameters for the method ots_hasCode: [address, defaultBlock].

- `params[0]` (string): The address.
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".

### Returns

<ResponseField name='result' type='boolean'>
  boolean indicating if the address contains a bytecode or not.
</ResponseField>

```bash Curl
curl --request POST \

  --header 'content-type: application/json' \
  --data '{

    "jsonrpc": "2.0",
    "method": "ots_hasCode",

    ]
  }'
```
