### Parameters

The parameters for method ots_searchTransactionsAfter: [address, defaultBlock, pageSize].

- `params[0]` (string): The address.
- `params[1]` (oneOf): Block identifier. Can be an integer block number or a string tag: "latest", "earliest", or "pending".
- `params[2]` (integer): The page size for the number of results to return.

### Returns

<ResponseField body='result' type='object'>
  Paginated list of transactions and receipts for a given address before or after a specified block number.
</ResponseField>
