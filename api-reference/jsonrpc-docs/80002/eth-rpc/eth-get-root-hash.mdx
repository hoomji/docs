---
title: 'eth_getRootHash'
openapi: 'POST /json-rpc'
playground: 'interactive'
---
### Parameters

Parameters: [fromBlock, toBlock]

- `params[0]` (integer): The starting block number (inclusive).
- `params[1]` (integer): The ending block number (inclusive).

### Returns

<ResponseField body='result' type='string'>
  Hex-encoded root hash covering the specified block range. Example: <code>"0x67518d9fc6bf9ff886ba1a4e932a0d0a8a1318b3f300518773aaf2210410cf73"</code>.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=80002 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_getRootHash",
    "params": [
      1000,
      1032
    ]
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_getRootHash",
  "params": [
    1000,
    1032
  ]
}
```

</RequestExample>
