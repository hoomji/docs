### Parameters

This method does not require any parameters.

### Returns

<ResponseField body='result' type='oneOf'>
  Returns syncing status. Either a sync status object if syncing, or `false` if not syncing.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=143 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "eth_syncing",
    "params": []
  }'
```

```json JSON-RPC
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "eth_syncing",
  "params": []
}
```

</RequestExample>
