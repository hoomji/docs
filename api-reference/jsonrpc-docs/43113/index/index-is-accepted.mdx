### Parameters

Parameters used to check the acceptance status of a container.

- `containerID` (string): The ID of the container to check.
- `encoding` (string): The encoding format of the request. Only "hex" is supported.

### Returns

<ResponseField name='result' type='object'>
  Boolean result indicating whether the container is accepted.
  <Expandable title='properties'>
    <ResponseField name='isAccepted' type='boolean'>
      True if the container is accepted in the index, false otherwise.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=43113 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "index.isAccepted",
    "params": {
      "containerID": "2dGz8JSfX17QqW12pARrt2iWNhXdcpXMYPVSYRn9HrReGpMXqQ",
      "encoding": "hex"
    }
  }'
```
