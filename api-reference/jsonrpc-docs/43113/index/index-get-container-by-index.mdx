### Parameters

Parameters to specify the container index and encoding format.

- `index` (integer): The sequential index of the container (starting from 0).
- `encoding` (string): The encoding format for the container data. Only "hex" is supported.

### Returns

<ResponseField name='result' type='object'>
  Details and content of the container at the specified index.
  <Expandable title='properties'>
    <ResponseField name='id' type='string'>
      The unique ID of the container.
    </ResponseField>
    <ResponseField name='bytes' type='string'>
      Hex-encoded byte representation of the container.
    </ResponseField>
    <ResponseField name='timestamp' type='string'>
      The timestamp (in Unix seconds) when the container was accepted.
    </ResponseField>
    <ResponseField name='index' type='string'>
      The index of the container (same as the request).
    </ResponseField>
    <ResponseField name='encoding' type='string'>
      The encoding used for the bytes field. Always "hex".
    </ResponseField>
  </Expandable>
</ResponseField>
