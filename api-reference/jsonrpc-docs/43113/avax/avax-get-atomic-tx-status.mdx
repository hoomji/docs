### Parameters

- `txID` (string): The transaction ID (in CB58 format).

### Returns

<ResponseField name='result' type='object'>
  The JSON-RPC result.
  <Expandable title='properties'>
    <ResponseField name='status' type='string'>
      The status of the atomic transaction.
    </ResponseField>
    <ResponseField name='blockHeight' type='string'>
      The block height in which the transaction was accepted. Only present if status is "Accepted".
    </ResponseField>
  </Expandable>
</ResponseField>
