### Parameters

- `addresses` (array): A list of X-Chain addresses to fetch UTXOs for.
- `limit` (integer): Optional maximum number of UTXOs to return. Defaults to 1024 if omitted or exceeded.
- `startIndex` (object): Pagination index from the previous response's endIndex.
- `sourceChain` (string): Optional source chain ID (e.g., "P" or "C"). Used when fetching atomic UTXOs.
- `encoding` (string): Encoding format for the returned UTXOs. Must be "hex" if specified.

### Returns

<ResponseField body='result' type='object'>
  The JSON-RPC result.
</ResponseField>
