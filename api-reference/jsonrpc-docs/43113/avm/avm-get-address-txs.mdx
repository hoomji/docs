### Parameters

Filter options to query transactions for an address and asset.

- `address` (string): The address for which to fetch balance-changing transactions.
- `assetID` (string): The asset ID or alias to filter transactions by.
- `cursor` (integer): Pagination offset (page number or record index). Optional.
- `pageSize` (integer): Number of transactions to return per page. Defaults to 1024.

### Returns

<ResponseField body='result' type='object'>
  List of transaction IDs affecting the address balance and the current pagination cursor.
</ResponseField>

<RequestExample>

```bash Curl
curl --request POST \
  --url https://api.uniblock.dev/uni/v1/json-rpc?chainId=43113 \
  --header 'content-type: application/json' \
  --data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "avm.getAddressTxs",
    "params": {
      "address": "X-avax19pm62y5n0yt76wgp8pd7jdhepmppc67y7yg0cq",
      "assetID": "AVAX",
      "pageSize": 20

  }'
```
