### Parameters

Returns the minimum amount of tokens required to validate or delegate on the given Subnet.

- `subnetID` (string): Optional ID of the subnet to query. Defaults to the Primary Network if omitted.

### Returns

<ResponseField body='result' type='object'>
  The JSON-RPC result.
</ResponseField>
