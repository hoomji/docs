To get started, you need to:

1. Visit [Uniblock Dashboard](https://dashboard.uniblock.dev).
2. Sign up using your Google account, GitHub account, or click the `Create an account` button to sign up with email, and complete the secure verification process.
3. Then afterwards you can, `Create a new project`. Head to our [Dashboard Quickstart Section](doc:dashboard-quickstart) for more details on how to configure and start using Uniblock!

### üì° Endpoints: JSON-RPC

JSON-RPC is a lightweight, stateless protocol used for remote procedure calls (RPCs). It is commonly used to interact with blockchain nodes, such as Ethereum. Uniblock supports JSON-RPC across a list of network and route your JSON-RPC requests to various integrated providers based on need. For more information head to this [section](doc:json-rpc-overview).
