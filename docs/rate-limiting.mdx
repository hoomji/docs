<Info>
  ### New Feature!

Note: The limits will depend on the throughput of your plans.

</Info>

We are introducing a dynamic way to manage API usage through Per Key Rate-Limiting, allowing users to assign specific rate limits to individual API keys within a single project. This feature provides aims to provide more flexible resource management.

## Get Started

1. Navigate to your project dashboard/overview page that displays your API keys. You can use the our navigation bar on the left side column for a direct link.

</Frame>

2. Click View All to go to the API Keys management page.

<Frame caption='API Keys Management Page'>
  <img src='/images/docs/7a6734c1e289dd78c6c6377fad0517420a855b46702233fd362956eda0c08faa-Screenshot_2025-01-23_at_4.35.55_PM.png' />
</Frame>

3. For the API Key that you want to set limits for through the limits column.
4. Set any limits you would like.

<Frame caption='Set Limits Popup'>
  <img src='/images/docs/80d2abb63320e55a07218eadbb9efc92d18642ee7fd023d5f4d531efdd1b8e8b-Screenshot_2025-01-23_at_4.36.42_PM.png' />
</Frame>

5. Wait 5-10 mins for our backend to update and start testing!
