# Prefix:

No prefix is required for LunarCrush, an example URL will therefore look like: `https://api.uniblock.dev/direct/v1/LunarCrush/path/to/endpoint`.

# Example:

<CodeGroup>
  ```bash bash
  curl --request GET \
    --url https://api.uniblock.dev/direct/v1/LunarCrush/public/nfts/list/v2?sort=percent_change_24h&limit=100&desc=1 \
    --header 'accept: application/json' \
    --header 'x-api-key: YOUR_UNIBLOCK_API_KEY'
  ```
</CodeGroup>
