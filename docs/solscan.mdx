For more information regarding the endpoints, please visit [SolScan's website](https://pro-api.solscan.io/pro-api-docs/v2.0).

# Prefix:

No prefix is required for Solscan, an example URL will therefore look like:

* Version 1 : `https://api.uniblock.dev/direct/v1/SolScan/v1.0/path/to/endpoint`.
* Version 2 : `https://api.uniblock.dev/direct/v1/SolScan/v2.0/path/to/endpoint`.

# Example:

<CodeGroup>
  ```bash bash
  curl --request GET \
    --url https://api.uniblock.dev/direct/v1/SolScan/v1.0/block/last \
    --header 'accept: application/json' \
    --header 'x-api-key: YOUR_UNIBLOCK_API_KEY'
  ```
</CodeGroup>

# Supported Endpoints:

## Version 1.0

### Block APIs

* **Block Last** **Description**: Fetch the latest block. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/block/last`

* **Block Transactions** **Description**: Retrieve transactions for a specific block. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/block/transactions`

* **Block Detail** **Description**: Get detailed information about a block. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/block/{block}`

### Transaction APIs

* **Transaction Last** **Description**: Fetch the most recent transaction. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/transaction/last`

* **Transaction Detail** **Description**: Retrieve details of a specific transaction by signature. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/transaction/{signature}`

### Account APIs

* **Account Tokens** **Description**: Fetch all tokens associated with an account. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/account/tokens`

* **Account Transactions** **Description**: Retrieve all transactions for a specific account. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/account/transactions`

* **Account Stake Accounts** **Description**: Retrieve staking accounts for a specific account. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/account/stakeAccounts`

* **Account SPL Transfers** **Description**: Fetch all SPL token transfers for a specific account. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/account/splTransfers`

* **Account SOL Transfers** **Description**: Retrieve SOL transfers for a specific account. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/account/solTransfers`

* **Account Export Transactions** **Description**: Export transaction history for a specific account. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/account/exportTransactions`

* **Account Export Rewards** **Description**: Export rewards history for a specific account. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/account/exportRewards`

* **Account Detail** **Description**: Retrieve details for a specific account. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/account/{account}`

### Token APIs

* **Token Holders** **Description**: Retrieve holders of a specific token. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/token/holders`

* **Token Meta** **Description**: Retrieve metadata for a specific token. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/token/meta`

* **Token Transfer** **Description**: Fetch transfer information for a specific token. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/token/transfer`

* **Token List** **Description**: Retrieve a list of tokens. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/token/list`

* **Market Token Detail** **Description**: Retrieve market details for a specific token. **URL**: `https://api.uniblock.dev/direct/v1/SolScan/v1.0/market/token/{tokenAddress}`
