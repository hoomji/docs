# Prefix:

No prefix is required for Birdeye, an example URL will therefore look like: `https://api.uniblock.dev/direct/v1/Birdeye/path/to/endpoint`.

# Example:

<CodeGroup>
  ```bash bash
  curl --request GET \
    --url https://api.uniblock.dev/direct/v1/Birdeye/defi/price?address=So11111111111111111111111111111111111111112 \
    --header 'accept: application/json' \
    --header 'x-api-key: YOUR_UNIBLOCK_API_KEY' \
    --header 'x-chain: solana'
  ```
</CodeGroup>
